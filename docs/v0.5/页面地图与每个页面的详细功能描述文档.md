# v0.5 页面地图与每个页面的详细功能描述文档

## 1. 文档概述

### 1.1 版本范围

`v0.5` 的界面范围以 `v0.3` 主面板和托盘能力为基础，新增 **独立设置窗口** 与 **平台能力降级提示**。

当前版本仍然 **不包含** 搜索页、收藏页、标签页和独立详情页。

### 1.2 v0.5 界面清单

| 编号 | 名称 | 类型 |
|------|------|------|
| P01 | 主面板 | 面板 |
| P02 | 设置窗口 | 窗口 |
| P02-1 | 通用设置页 | 窗口子页 |
| P02-2 | 记录与存储页 | 窗口子页 |
| P02-3 | 快捷键设置页 | 窗口子页 |
| P02-4 | 隐私设置页 | 窗口子页 |
| P04 | 托盘菜单 | 系统菜单 |
| P05 | 通用确认弹窗 | 弹窗 |
| V01 | 文本卡片 | 视图组件 |
| V02 | 图片卡片 | 视图组件 |
| V03 | 文件卡片 | 视图组件 |
| V06 | 平台能力提示条 | 视图组件 |
| V07 | 快捷键录制器 | 视图组件 |
| V08 | 黑名单规则行 | 视图组件 |

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 触发方式 | 全局快捷键、托盘菜单 |
| 关闭方式 | `Esc`、托盘隐藏、粘贴后自动关闭 |
| 位置 | 当前活动显示器底部 `work_area` |
| 视觉风格 | 无边框、半透明、始终置顶 |

### 3.2 v0.5 变化点

- 保留 `v0.3` 的横向卡片流与数字键快选
- 新增“打开设置”入口按钮
- 新增主题联动，支持浅色 / 深色 / 跟随系统
- 在能力降级环境下，可显示 `V06` 平台能力提示条

### 3.3 区域说明

| 区域 | 说明 |
|------|------|
| 顶部状态区 | 监听暂停提示、平台能力提示 |
| 卡片列表区 | 横向滚动历史卡片 |
| 底部快捷键栏 | `Enter` / `Shift+Enter` / `Delete` / `1-9` / `Esc` |
| 设置入口区 | 打开设置窗口 |

### 3.4 状态变体

| 状态 | 说明 |
|------|------|
| 正常态 | 有记录、监听正常 |
| 暂停监听态 | 顶部显示弱提示 |
| 降级提示态 | 当前平台能力部分不可用，显示 `V06` |
| 空状态 | 无记录时显示空态视图 |

---

## 4. P02 设置窗口（Settings Window）

### 4.1 基本信息

| 项目 | 说明 |
|------|------|
| 类型 | 独立原生窗口 |
| 打开方式 | 托盘菜单、主面板按钮 |
| 窗口策略 | 单实例；重复打开时激活并置前 |
| 布局 | 左侧导航 + 右侧内容 |

### 4.2 布局结构

```text
┌─────────────────────────────────────────────┐
│ 设置                                        │
├──────────────┬──────────────────────────────┤
│ 通用         │ 标题 / 描述                   │
│ 记录与存储   │ 表单内容区                    │
│ 快捷键       │ 保存反馈 / 能力提示           │
│ 隐私         │                               │
└──────────────┴──────────────────────────────┘
```

### 4.3 公共行为规格

- 切换分组时保留当前已保存状态
- 有未保存改动时关闭窗口需弹出 `P05` 确认
- 保存成功后显示轻量 Toast，不强制关闭窗口
- 平台能力受限时，相关控件禁用并解释原因

## 5. P02-1 通用设置页

### 5.1 功能定位

承载全局基础偏好：

- 主题模式
- 语言占位（当前版本只读或固定中文）
- 开机自启动
- 当前平台能力概览

### 5.2 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| 主题模式 | 单选 | `浅色 / 深色 / 跟随系统` |
| 开机自启动 | 开关 | 同步系统级启动项 |
| 当前平台 | 只读 | `macOS / Windows / Linux` |
| 会话能力说明 | 提示块 | 显示降级项与建议 |

### 5.3 边界情况

- Wayland 下自启动可用但全局快捷键不可用时，必须分项显示
- 切换主题后不需要重新打开主面板即可见效

---

## 6. P02-2 记录与存储页

### 6.1 功能定位

管理保留策略与本地资源占用。

### 6.2 字段说明

| 字段 | 说明 |
|------|------|
| 文本记录上限 | 限制文本历史数量 |
| 图片记录上限 | 限制图片历史数量 |
| 文件记录上限 | 限制文件历史数量 |
| 图片存储空间上限 | 限制图片原图与缩略图总空间 |
| 记录图片 | 是否采集图片 |
| 记录文件 | 是否采集文件 |

### 6.3 行为规格

- 保存后如新上限低于当前已有数据，后台触发补偿清理
- 输入值越界时，不允许保存并给出字段级错误
- 关闭图片采集不应删除已有图片历史，只影响后续采集

---

## 7. P02-3 快捷键设置页

### 7.1 功能定位

管理调出主面板的全局快捷键。

### 7.2 布局结构

| 区域 | 说明 |
|------|------|
| 当前快捷键 | 显示已生效组合 |
| 录制区 | 使用 `V07` 录制新的组合键 |
| 校验反馈区 | 冲突、格式、平台支持状态 |
| 恢复默认按钮 | 一键回到平台默认值 |

### 7.3 行为规格

- 录制时阻断普通表单输入，直接捕获组合键
- 若检测到冲突，不允许保存
- Wayland 下全局快捷键不可用时，整个保存动作禁用并显示能力提示

---

## 8. P02-4 隐私设置页

### 8.1 功能定位

管理黑名单规则与高风险操作入口。

### 8.2 内容组成

| 区块 | 说明 |
|------|------|
| 黑名单规则列表 | 展示已保存规则，使用 `V08` |
| 新增规则表单 | 添加应用名称、平台、标识符、匹配类型 |
| 清空历史入口 | 触发 `P05` 高风险确认弹窗 |

### 8.3 行为规格

- 同一平台 + 同一标识符不可重复添加
- 停用规则不会删除，仅不参与匹配
- 删除规则后不追溯恢复历史记录

---

## 9. P04 托盘菜单（Tray Menu）

### 9.1 菜单项说明

| 菜单项 | 类型 | 说明 |
|--------|------|------|
| 显示主面板 / 隐藏主面板 | 普通项 | 根据当前窗口状态动态变化 |
| 打开设置 | 普通项 | 激活设置窗口 |
| 暂停监听 / 恢复监听 | 普通项 | 与运行态同步 |
| 开机自启动 | 勾选项 | 与设置页同步 |
| 清空历史 | 危险项 | 触发确认弹窗 |
| 打开日志目录 | 普通项 | 排查问题 |
| 退出应用 | 危险项 | 退出整个应用 |

### 9.2 v0.5 变化点

- 新增“打开设置”一级入口
- 与设置页共享 `launch_at_login` 状态
- 在能力受限平台可为不可用项展示禁用或说明文案

---

## 10. P05 通用确认弹窗（Confirm Dialog）

### 10.1 使用场景

- 清空历史
- 关闭有未保存改动的设置窗口

### 10.2 交互要求

- 默认焦点在“取消”按钮
- 高风险确认按钮使用危险色
- 弹窗关闭后焦点返回原触发控件

---

## 11. 视图组件说明

### 11.1 V06 平台能力提示条

- 用于显示 `Wayland`、权限不足、平台注册失败等降级信息
- 出现时不遮挡核心设置项标题
- 文案需包含“当前限制 + 建议替代路径”

### 11.2 V07 快捷键录制器

- 空闲态显示当前值
- 录制态显示“请按下新的组合键”
- 冲突态显示错误文案与不可保存原因

### 11.3 V08 黑名单规则行

- 展示应用名、平台、匹配标识符、启用状态
- 支持启停开关与删除操作

---

## 12. 页面间状态转换

```text
托盘 / 主面板 → 打开设置 → 读取设置快照 → 修改配置
                                     ├── 保存成功 → 广播 settings-updated
                                     └── 关闭未保存 → P05 确认弹窗
```

---

## 13. 可访问性要求

- 设置窗口所有表单控件支持键盘导航
- 快捷键录制器必须有明确的屏幕阅读器状态说明
- 颜色不是唯一反馈来源，冲突和禁用态需同时依赖图标 / 文案
- 主题切换后对比度应保持可读
