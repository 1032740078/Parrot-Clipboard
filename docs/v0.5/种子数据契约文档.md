# v0.5 种子数据契约文档

## 1. 文档概述

### 1.1 版本范围

本文档定义 **v0.5 Beta 版本** 的种子数据与测试夹具契约，用于：

- 设置窗口 UI 开发演示
- 快捷键冲突校验测试
- 黑名单过滤测试
- 三平台能力矩阵与降级提示测试
- 配置迁移回归测试

### 1.2 设计原则

- 使用脱敏应用名与路径，不出现真实隐私内容
- 同一组夹具只覆盖单一主题，避免语义混杂
- 种子数据必须可重复加载、可重置

| Linux X11 | `Shift+Ctrl+V` | 通过 |

## 3. 类型覆盖要求

### 3.1 设置覆盖

- 三种主题模式都要有夹具
- 自启动开 / 关都要覆盖
- 记录数量和存储上限要覆盖最小值、正常值、越界值

### 3.2 黑名单覆盖

- 新增规则
- 重复规则
- 禁用规则
- 多平台不同匹配类型规则

### 3.3 平台能力覆盖

- macOS 完整支持
- Windows 完整支持
- Linux X11 完整支持
- Linux Wayland 降级支持

## 4. 资产文件契约

### 4.1 推荐目录

```text
src/__tests__/fixtures/
├── settings/
│   ├── settings.snapshot.json
│   ├── legacy-config.v1.json
│   ├── shortcut-conflicts.json
│   └── blacklist-rules.json
└── platform/
    └── capabilities.json
```

### 4.2 路径约束

- 不写入真实用户目录
- 不引用真实应用路径
- 文件名尽量使用稳定英文，展示文案放在 JSON 字段中

---

## 5. 加载约定

### 5.1 开发模式加载

- 设置窗口 Story / 测试页优先加载 `SEED-501`
- 平台能力提示演示优先加载 `SEED-505`

### 5.2 测试模式加载

- 配置迁移测试加载 `SEED-502`
- 快捷键校验测试加载 `SEED-503`
- 黑名单过滤测试加载 `SEED-504`

---

## 6. 覆盖的测试场景

| 场景 | 对应种子 |
|------|----------|
| 设置窗口首屏回显 | `SEED-501` |
| `v0.3 -> v0.5` 配置迁移 | `SEED-502` |
| 快捷键冲突 / 不支持判断 | `SEED-503` |
| 黑名单命中过滤 | `SEED-504` |
| X11 / Wayland 能力提示 | `SEED-505` |

---

## 7. 数据清理约定

- 每条测试用例执行后重置配置文件与运行态
- 黑名单命中过滤测试后清空临时历史记录
- 不允许把真实系统快捷键冲突信息固化为无法复现的本地夹具结果
