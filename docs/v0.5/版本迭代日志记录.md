## 3. 当前版本结论

### 已锁定事项

- `v0.5` 不提前引入搜索、收藏、云同步、导入导出
- `v0.5` 新增设置窗口、自定义快捷键、黑名单规则、主题与容量策略
- `Linux Wayland` 采用“能力探测 + 明确降级提示”，不做假支持
- `config.json` 升级为分组化、版本化结构，历史记录 SQLite 结构保持不变
- 配置损坏时允许自动回退默认值并原地自愈，优先保证应用可启动
- 设置修改统一经 `SettingsProfile` 聚合与校验服务处理，避免命令层重复实现规则
- 平台能力统一由 `PlatformCapabilityResolver` 输出，Wayland 与未知会话采用保守降级策略
- 设置窗口已具备单实例容器与独立渲染入口，第 1 批基础设施已完成
- Windows 运行时已切换到真实平台实现，第 2 批跨平台落地已启动
- Linux X11 运行时已切换到真实平台实现，Wayland / 未知会话已切入真实降级策略
- 托盘和设置窗口壳层已经直接消费平台能力矩阵，第 2 批跨平台能力落地完成
- 设置窗口已完成导航壳层与未保存确认，为后续表单接入准备好稳定骨架

### 下一步建议

1. 继续完成通用页与记录页表单，接入真实草稿与保存逻辑
2. 然后补齐快捷键页、隐私页和设置相关 IPC / Event
3. 后续完成黑名单过滤联动与自动化回归
