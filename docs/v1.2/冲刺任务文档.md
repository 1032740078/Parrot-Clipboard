## 4. 风险与缓解

| 风险                   | 说明                                     | 缓解方案                                       |
| ---------------------- | ---------------------------------------- | ---------------------------------------------- |
| 双击触发两次粘贴       | 浏览器双击与单击事件叠加可能引发重复请求 | 引入防重入状态，双击路径只允许一次真实提交     |
| 可视编号语义混乱       | 用户可能把编号理解成全局绝对顺序         | 明确“当前可视区域”提示，并统一徽标与快捷键映射 |
| 图片原图回退过重       | 大图直接作为预览源可能带来额外加载成本   | 只对可见卡片按需回退，不做全量详情拉取         |
| 设置页改版引发认知断裂 | 原有分组内容位置变化                     | 保持分组名称不变，仅调整布局与能力分组归位     |
