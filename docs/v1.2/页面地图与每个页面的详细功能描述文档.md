# v1.2 页面地图与每个页面的详细功能描述文档

## 1. 文档概述

### 1.1 版本范围

`v1.2` 不新增独立窗口，主要调整 **主面板** 与 **设置窗口** 的交互与信息架构；其他页面沿用 `v1.1` 能力，仅补充与当前版本的关系说明。

### 1.2 v1.2 界面清单

| 页面 / 视图 | 是否变更 | 说明 |
|-------------|----------|------|
| `P01` 主面板 | 是 | 本版核心改动页 |
| `P02` 设置窗口 | 是 | 布局与分组结构调整 |
| `P03` 关于页 | 否 | 结构沿用 `v1.0 / v1.1` |
| `P04` 权限引导弹窗 | 小幅关联 | 双击粘贴失败时可作为降级入口 |
| `P05` 托盘菜单 | 否 | 无信息架构变化 |
| `P06` 通用确认弹窗 | 否 | 无结构变化 |

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 页面定位 | 高频复制 / 粘贴的主工作界面 |
| 打开方式 | 全局快捷键、托盘入口 |
| 关闭方式 | `Esc`、粘贴成功、失去焦点、显式隐藏 |
| 本版优先级 | `P0` |

### 3.2 v1.2 变化点

- 新增鼠标单击选中与双击直接粘贴
- 图片卡片预览改为支持缩略图、原图回退与占位态
- 卡片右上角 `1~9` 编号改为依据当前可视区域动态变化
- `1~9` 与 `Command + 1~9` 均改为操作当前可视区域槽位

### 3.3 区域说明

| 区域 | 说明 |
|------|------|
| 标题区 | 展示最近记录与当前说明 |
| 卡片列表区 | 横向卡片列表，负责高亮、鼠标交互、滚动与可视槽位计算 |
| 快捷编号区 | 位于卡片右上角，显示当前可视槽位 `1~9` |
| 快捷键提示区 | 展示 `Enter`、`Shift+Enter`、`1~9`、`Command+1~9`、`Esc` |

### 3.4 状态变体

| 状态 | 描述 |
|------|------|
| 键盘选中态 | 通过左右键或数字键切换，保持既有高亮 |
| 鼠标选中态 | 单击卡片后进入，与键盘选中视觉一致 |
| 预览正常态 | 图片卡片展示真实缩略图或原图回退 |
| 预览占位态 | 图片资源不可用时展示占位说明 |
| 长列表态 | 横向滚动后重新计算可视槽位编号 |
| 权限受限态 | 允许浏览与选中，不允许实际粘贴 |

### 3.5 关键交互规格

| 交互 | 结果 |
|------|------|
| 鼠标单击卡片 | 仅选中，不粘贴 |
| 鼠标双击卡片 | 直接粘贴当前卡片 |
| `1~9` | 快选当前可视槽位卡片 |
| `Command+1~9` | 快贴当前可视槽位卡片 |
| 横向滚动 | 更新当前可视槽位起点与编号 |
| 图片缩略图更新事件 | 自动刷新卡片预览 |

## 4. P02 设置窗口（Settings Window）

### 4.1 功能定位

- 延续既有设置保存与校验能力
- 本版主要调整**导航结构**与**会话能力信息承载方式**

### 4.2 v1.2 布局结构

| 区域 | 说明 |
|------|------|
| 左侧导航区 | 固定展示分组入口与当前分组高亮 |
| 右侧内容区 | 展示当前分组的标题、描述、表单与说明 |
| 顶部状态区 | 展示当前分组、未保存状态与页面级操作 |

### 4.3 分组清单

| 分组 | 说明 |
|------|------|
| `P02-1` 通用 | 主题、语言、开机自启动 |
| `P02-2` 记录与存储 | 历史保留与采集范围 |
| `P02-3` 快捷键 | 调出面板快捷键录制与校验 |
| `P02-4` 隐私 | 黑名单规则管理 |
| `P02-5` 会话能力 | 当前平台 / 会话支持状态与降级说明 |

### 4.4 行为规格

- 左侧导航切换分组时，仅右侧内容区变化
- 当前分组标题与描述在右侧顶部固定展示
- 未保存拦截、保存按钮与 Toast 逻辑沿用既有实现
- “会话能力”分组独立承载完整支持与受限说明，不再混入通用页主体

---

## 5. P02-5 会话能力分组（Capability Section）

### 5.1 功能定位

用于集中展示：

- 当前平台
- 当前会话类型
- 当前能力矩阵
- 完整支持或降级原因
- 替代路径说明

### 5.2 内容组成

| 模块 | 内容 |
|------|------|
| 摘要卡片 | `当前会话能力完整支持` 或 `当前会话能力受限` |
| 能力矩阵 | 粘贴板监听、全局快捷键、自启动、托盘、活动应用识别 |
| 说明区 | 受限原因与替代操作建议 |

### 5.3 状态变体

- **完整支持态**：展示绿色成功摘要与全支持能力矩阵
- **受限态**：展示黄色 / 红色摘要与受限原因列表
- **加载失败态**：展示读取失败说明与重试建议

## 7. P04 权限引导弹窗（Permission Guide）

### 7.1 触发条件

- `macOS` 辅助功能权限缺失
- 用户通过双击卡片、`Enter` 或 `Command + 数字` 尝试执行粘贴

### 7.2 行为规格

- 弹窗展示“为什么需要权限”与“如何授权”
- 关闭弹窗后主面板保持可见，允许用户继续浏览记录

---

### 8.1 菜单项说明

- 关于
- 暂停 / 恢复监听
- 清空历史
- 退出应用

### 8.2 与 v1.2 的关系

- 无信息架构变化
- 继续作为设置页与主面板的重要兜底入口

---

## 9. 视图组件说明

### 9.1 `V21` 可视槽位编号

- 位于卡片右上角
- 只展示当前可视区域槽位 `1~9`
- 横向滚动后自动重排

### 9.2 `V22` 图片预览回退态

- 优先显示缩略图
- 次选显示原图回退
- 最终降级为占位态

### 9.3 `V23` 会话能力摘要卡片

- 独立存在于“会话能力”分组中
- 显示完整支持或受限摘要

---

## 10. 页面间状态转换

- 主面板双击粘贴成功 → 主面板隐藏 → 焦点回到目标应用
- 主面板粘贴失败 → 保持主面板打开 → 可进入权限引导弹窗
- 设置窗口切换到“会话能力” → 读取当前能力快照 → 渲染独立能力摘要卡片

---

## 11. 可访问性要求

- 卡片单击与双击必须保留键盘等价路径，不以鼠标交互替代键盘主链路
- 可视槽位编号需保持足够对比度，便于识别
- 会话能力分组的完整支持 / 受限状态不能只靠颜色区分，需配合明确文案
