## 2. 本次变更明细

### 2.1 2026-03-09 `v1.2-main-panel-pointer`

**变更类型**：`feat(main-panel-pointer)`

**主要变更**：

- 新增 `src/hooks/recordPaste.ts`，把键盘粘贴与鼠标双击粘贴统一收敛到同一条执行链路
- 更新 `src/components/MainPanel/index.tsx` 与 `src/components/MainPanel/CardList.tsx`，让卡片支持鼠标单击选中、双击直贴
- 更新 `src/hooks/useKeyboard.ts`，复用共享粘贴执行能力，保证键盘 / 鼠标权限校验与失败提示一致
- 新增与更新 `src/__tests__/components/MainPanel.test.tsx`、`src/__tests__/components/MainPanel.permission.test.tsx`，覆盖单击选中、双击粘贴、防重复提交与权限缺失场景

**产出结果**：

- 主面板不再局限于纯键盘操作，鼠标用户可直接点击接管选中态并通过双击完成粘贴
- 双击导致的重复请求风险被前端防重入锁收敛，鼠标与键盘粘贴行为保持一致
- 权限缺失、错误提示与面板隐藏行为统一到同一条运行态逻辑，后续扩展更稳定

**验证结果**：

- 已执行 `pnpm exec vitest run src/__tests__/components/MainPanel.test.tsx src/__tests__/components/MainPanel.permission.test.tsx src/__tests__/hooks/useKeyboard.test.tsx src/__tests__/hooks/useKeyboard.permission.test.tsx`
- 已执行 `pnpm run type-check`
- 已执行 `pnpm run lint`
