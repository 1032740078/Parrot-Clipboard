# v1.4 状态码契约文档

## 1. 文档概述

### 1.1 版本范围

本文档仅描述 `v1.4` 为支持搜索、类型扩展、文本预览查找替换与多屏定位所涉及的状态码契约。

### 1.2 错误对象格式

```ts
interface IpcError {
  code: ErrorCode;
  message: string;
  details?: Record<string, unknown>;
}
```

### 1.3 错误传递机制

- Tauri Command 失败时统一返回结构化 `IpcError`
- 前端搜索、筛选和预览工作区先处理可恢复错误，再决定是否上抛 Toast
- 多屏定位与来源应用图标解析失败优先走降级逻辑，不应默认中断主流程

## 3. 状态码详细说明

### SC-001：`INVALID_PARAM`

- 适用场景：`limit <= 0`、`type_filter` 非法、`replace_text` 参数不完整
- 前端处理：阻止继续请求，并提示用户修正输入

### SC-002：`RECORD_NOT_FOUND`

- 适用场景：搜索结果中目标记录已被删除，或预览窗口打开时目标记录不存在
- 前端处理：关闭局部视图并刷新列表

### SC-003：`CLIPBOARD_WRITE_ERROR`

- 适用场景：粘贴或文本替换保存后的写回失败
- 前端处理：保留当前窗口状态并提示失败

### SC-004：`KEY_SIM_ERROR`

- 适用场景：系统模拟粘贴按键失败
- 前端处理：提示系统权限或环境问题

### SC-005：`DB_ERROR`

- 适用场景：搜索、迁移、更新文本、重建分类字段时数据库异常
- 前端处理：搜索与预览进入局部错误态，保留主面板

### SC-006：`FILE_ACCESS_ERROR`

- 适用场景：文件详情、文稿路径、图标缓存或媒体资产不可访问
- 前端处理：展示降级摘要，不阻断列表

### SC-007：`IMAGE_PROCESS_ERROR`

- 适用场景：图片缩略图、原图读取或 OCR 相关流程异常
- 前端处理：以错误占位或文本提示降级

### SC-008：`WINDOW_ERROR`

- 适用场景：预览窗口、设置页、关于页打开失败或定位失败
- 前端处理：优先尝试一次安全回退定位，再提示用户

### SC-009：`UNSUPPORTED_PLATFORM_FEATURE`

- 适用场景：某平台不支持背景模糊、来源应用图标解析或显示器定位能力
- 前端处理：应用降级样式或安全默认行为

### SC-010：`INTERNAL`

- 适用场景：未被更细状态码覆盖的异常
- 前端处理：提示通用错误并记录日志

### SC-011：`SEARCH_QUERY_INVALID`

- 适用场景：搜索关键字过长、包含非法控制字符或超出系统限制
- 前端处理：阻止查询并在搜索框附近提示

### SC-012：`CLASSIFICATION_ERROR`

- 适用场景：新增记录无法完成语义分类，或旧记录重分类失败
- 前端处理：回退到最安全的 `text / image / files` 类型，并记录日志

### SC-013：`DISPLAY_RESOLVE_ERROR`

- 适用场景：主面板当前屏幕无法识别、显示器信息失效或返回空工作区
- 前端处理：预览窗口降级到主屏或默认位置

### SC-014：`PREVIEW_REPLACE_ERROR`

- 适用场景：文本查找替换规则无效、替换结果保存失败或目标内容不允许替换
- 前端处理：保留原文，提示替换失败原因

## 5. 前端统一处理约定

- `SC-011` 应在搜索框内联反馈，不需要全局 Toast
- `SC-012` 与 `SC-013` 优先走降级，不把轻微能力缺失变成阻断错误
- `SC-014` 只影响文本预览工作区，不应把主面板一起关闭

---

## 6. 成功响应约定

- 搜索成功时，即使结果为空也返回 `200` 语义结果和空数组，而不是错误
- 预览窗口定位成功时，可附带最终解析得到的显示器信息供前端记录
- 文本替换成功后返回最新详情对象，用于同步搜索索引与预览内容
