## 1. 文档概述

### 1.1 版本范围

本文档仅描述 `v1.4` 当前版本的页面地图与页面 / 视图功能说明，聚焦主面板信息架构重构、搜索与筛选、新类型卡片、文本预览编辑器、多窗口毛玻璃风格和多屏定位。

### 1.2 `v1.4` 界面清单

| 页面 / 视图 | 是否有变化 | 说明 |
|-------------|------------|------|
| `P01` 主面板（Main Panel） | 是 | 本版本核心变化承载页 |
| `P02` 预览窗口（Preview Window） | 是 | 支持多屏定位与文本编辑器升级 |
| `P03` 设置窗口（Settings Window） | 是 | 升级为毛玻璃半透明风格 |

| `P05` 权限引导窗口（Permission Guide） | 否 | 沿用既有逻辑 |

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 功能定位 | 浏览、搜索、筛选、预览、粘贴和删除剪贴板记录 |
| 打开方式 | 全局快捷键、托盘菜单、窗口显隐事件 |
| 关闭方式 | `Esc`、失焦、粘贴成功后自动隐藏 |
| 核心变化 | 顶部居中搜索、左侧筛选栏、主面板毛玻璃升级、关于入口移除 |

### 3.2 `v1.4` 变化点

- 顶部去掉无效说明文案，改为横向居中的动态搜索框
- 左侧新增竖向类型筛选栏
- 主面板右侧关于入口移除
- 卡片扩展为七类并引入类型配色、来源应用图标与新底部信息布局
- 底部快捷提示整体居中并限制最大宽度

### 3.3 区域说明

| 区域 | 说明 |
|------|------|
| 左侧筛选栏 | `全部 / 文本 / 图片 / 文件 / 超链接 / 视频 / 音频 / 文稿` |
| 顶部搜索区 | 动态宽度搜索框、清空按钮 |
| 卡片列表区 | 七类卡片统一列表区域 |
| 底部快捷提示区 | 居中显示快捷键提示，限制最大宽度 |
| 浮层区 | Toast、上下文菜单等辅助浮层 |

### 3.4 状态变体

| 状态 | 说明 |
|------|------|
| 默认态 | 无搜索词，显示当前类型或全部列表 |
| 搜索态 | 有搜索词，结果实时刷新 |
| 筛选态 | 左侧某类型被选中 |
| 空结果态 | 搜索或筛选后无匹配卡片 |
| 执行动作态 | 粘贴、删除、预览打开中 |

### 3.5 关键交互规格

- 输入搜索词后自动去抖并刷新结果
- 点击清空图标后回到空查询
- 切换左侧类型后保留当前搜索词
- 右键菜单、快捷键和预览入口继续保留
- `Shift+Enter` 在 `payload_type = files` 的记录上统一输出路径列表，因此 `视频 / 音频 / 文稿 / 文件` 语义类型共享同一条路径纯文本粘贴语义

## 4. P02 预览窗口（Preview Window）

### 4.1 功能定位

- 用于查看完整内容
- 文本类型进一步升级为轻量编辑器，支持代码高亮、搜索、替换
- 打开时优先跟随主面板当前所在屏幕

### 4.2 内容组成

| 区域 | 说明 |
|------|------|
| 头部极简操作区 | 仅保留必要标题、类型信息或简化操作控件 |
| 文本编辑器区 | 代码高亮、搜索、替换、自动保存 |
| 媒体预览区 | 图片、视频、音频、文稿的主体预览区域 |
| 状态提示区 | 保存中、保存失败、无法预览等局部状态 |

### 4.3 状态变体

| 状态 | 说明 |
|------|------|
| Loading | 正在加载详情 |
| Ready | 内容已可见 |
| Editor Search Active | 文本查找面板已展开 |
| Saving | 文本替换后正在保存 |
| Error | 详情或保存失败 |

### 4.4 行为规格

- 文本类型显示代码高亮并可编辑
- 支持搜索、替换、全部替换
- 打开预览时优先定位到主面板所在屏幕
- 屏幕定位失败时回退为系统默认居中
- `视频 / 音频 / 文稿` 记录在预览中按各自语义展示，但纯文本粘贴仍回到文件路径列表语义

## 5. P03 设置窗口（Settings Window）

### 5.1 功能定位

- 提供设置项查看与配置能力
- `v1.4` 不新增复杂配置组，但视觉升级为毛玻璃风格

### 5.2 `v1.4` 变化点

- 背景升级为半透明毛玻璃
- 保持既有表单逻辑不变
- 与主面板、预览窗口共享统一窗口视觉语言

---

## 6. P04 关于窗口（About Window）

### 6.1 功能定位

- 提供应用基础信息与版本信息

### 6.2 `v1.4` 变化点

- 内容精简，只保留版本、平台、基础说明和必要链接
- 背景升级为毛玻璃半透明风格
- 从主面板移除入口，仅保留托盘等次级入口

---

## 7. P05 权限引导窗口（Permission Guide）

### 7.1 触发条件

- 粘贴或系统控制因权限受限而失败

### 7.2 与 `v1.4` 的关系

- 信息架构不变
- 视觉上可与全局窗口风格对齐，但不作为本版核心改动

---

## 8. P06 托盘菜单（Tray Menu）

### 8.1 菜单项说明

- 打开主面板
- 打开设置窗口
- 打开关于窗口
- 退出应用

### 8.2 与 `v1.4` 的关系

- 托盘继续承接关于入口
- 菜单结构原则上不扩张

## 9. 视图组件说明

### 9.1 `V41` 顶部搜索框

- 定位：主面板顶部中心区域
- 功能：模糊搜索全部卡片内容
- 特性：动态宽度、内置清空按钮

### 9.2 `V42` 左侧类型筛选栏

- 定位：主面板左侧固定竖向栏
- 功能：快捷筛选七类记录
- 特性：高亮当前筛选项，支持与搜索组合

### 9.3 `V43` 来源应用图标

- 定位：卡片标题栏右侧
- 功能：展示复制来源应用
- 特性：支持真实图标与占位图标回退

### 9.4 `V44` 文本预览编辑器工具区

- 定位：预览窗口内文本内容区顶部或悬浮工具条
- 功能：代码高亮、查找、替换
- 特性：与自动保存协同

### 9.5 `V45` 底部快捷提示条

- 定位：主面板底部
- 功能：展示高频快捷键
- 特性：整体居中，限制最大宽度

---

## 10. 页面间状态转换

- `P01 -> P02`：用户打开记录预览
- `P02 -> P01`：用户关闭预览窗口
- `P01 -> P03`：用户打开设置窗口
- `P06 -> P04`：用户从托盘打开关于窗口
- `P01 -> P05`：粘贴权限受限触发权限引导
- `P02 -> P01`：文本替换保存后，主面板与搜索结果同步刷新

---

## 11. 可访问性要求

- 左侧筛选栏每一项都有明确可点击区域
- 毛玻璃背景下仍需保证足够对比度
- 搜索框清空按钮、预览查找 / 替换控件需可见且可键盘访问
- 来源应用图标失败时必须有可理解占位，不可留下破图
