# v1.3 用户故事文档

## 1. 文档概述

### 1.1 版本范围

本文档仅描述 **v1.3** 当前版本的新增与调整能力。

`v1.3` 默认继承 `v1.2` 已完成的卡片鼠标选中、双击粘贴、图片预览修复、可视区域快捷编号、设置页双栏布局与会话能力分组；本版本只聚焦以下 3 项体验升级：

- 选中的卡片支持通过 **空格键** 打开“**居中放大预览**”，用于查看完整内容
- 卡片支持 **右键选中并弹出上下文菜单**，通过鼠标完成预览、粘贴、纯文本粘贴、删除等操作
- 主面板与卡片样式升级为 **半透明、玻璃拟态、科技感** 的视觉风格，提升高级感与辨识度

以下能力 **不属于 `v1.3` 当前版本范围**：

- 全文搜索、标签、收藏、置顶、批量操作
- 新增 SQLite 业务表、数据库迁移、配置结构升级
- 自定义右键菜单项、用户可配置的透明度 / 模糊度 / 动效参数
- 新的系统级全局快捷键方案、拖拽排序、跨端同步
- OCR、AI 分类、卡片分组与详情独立窗口

### 1.2 角色定义

| 角色 | 描述 | 主要诉求 |
|------|------|----------|
| 键盘高频用户 | 习惯使用快捷键浏览和操作剪贴板记录的用户 | 不离开键盘即可快速查看完整内容 |
| 鼠标操作用户 | 更偏向鼠标右键、菜单式交互的桌面用户 | 不记快捷键也能完成主要操作 |
| 视觉敏感用户 | 注重界面质感、可读性与信息层级的用户 | 面板更现代、更清晰、更有科技感 |

## 3. 用户故事详细说明

### E01 - 空格聚焦预览

#### US-01-01：按下空格后打开当前选中卡片的居中预览

**作为** 通过键盘浏览剪贴板历史的用户  
**我希望** 在选中卡片后按下空格键，就能在屏幕中心看到该卡片的放大预览  
**以便** 我在不执行粘贴的前提下，先快速确认完整内容。

**验收要点**：
- 仅对“当前已选中卡片”生效，不改变既有选中逻辑
- 预览层在屏幕中心显示，背景带遮罩，不影响主面板原始顺序
- 打开预览不会触发粘贴、删除或隐藏主面板
- 当没有选中卡片时，按空格应静默忽略

#### US-01-02：预览层可以查看完整内容并方便关闭

**作为** 需要仔细核对记录内容的用户  
**我希望** 预览层中能看到完整文本、完整图片或完整文件信息，并能用自然方式退出  
**以便** 我可以高效确认内容，而不是被强制跳转到其他窗口。

**验收要点**：
- 文本记录显示完整文本内容，支持滚动查看长内容
- 图片记录显示大图预览；文件记录显示完整文件列表、数量和路径摘要
- 支持 `Esc`、再次按空格、点击遮罩关闭预览
- 关闭预览后，主面板保持原选中项不变

### E02 - 右键上下文菜单

#### US-02-01：右键卡片时自动选中并弹出菜单

**作为** 更习惯鼠标操作的用户  
**我希望** 右键某张卡片时，该卡片先被选中，再弹出菜单  
**以便** 菜单动作和当前操作对象保持一致，不会误操作其他卡片。

**验收要点**：
- 右键时先同步更新选中态，再显示菜单
- 菜单锚点优先靠近鼠标点击位置，同时避免超出面板边界
- 同一时刻只允许存在一个卡片上下文菜单
- 点击菜单外区域、按 `Esc` 或执行完成后，菜单自动关闭

#### US-02-02：通过菜单执行主要操作

**作为** 不熟悉快捷键的用户  
**我希望** 通过右键菜单直接完成预览、粘贴、纯文本粘贴、删除  
**以便** 只靠鼠标也能完成高频操作。

**验收要点**：
- 菜单至少包含：`预览完整内容`、`直接粘贴`、`纯文本粘贴（文本卡片）`、`删除记录`
- `纯文本粘贴` 仅对文本卡片启用；图片 / 文件卡片显示禁用态或不展示
- `直接粘贴`、`纯文本粘贴` 复用既有 `paste_record` 链路
- `删除记录` 复用既有 `delete_record` 链路，成功后列表立即更新

#### US-02-03：菜单反馈清晰，避免误触

**作为** 在高频操作中使用鼠标菜单的用户  
**我希望** 危险动作和普通动作在视觉上有清晰分层  
**以便** 减少误删或误粘贴。

**验收要点**：
- `删除记录` 采用危险色与独立分组样式
- 菜单 hover、禁用、执行中状态清晰可见
- 粘贴失败时保留主面板并给出错误提示
- 连续右键不同卡片时，菜单内容应跟随当前卡片实时刷新

### E03 - 主面板视觉升级

#### US-03-01：主面板背景升级为半透明玻璃质感

**作为** 对桌面应用观感敏感的用户  
**我希望** 主面板背景具备半透明、模糊、柔和边框和科技光感  
**以便** 产品看起来更现代，不再像普通纯色浮层。

**验收要点**：
- 主面板采用半透明背景，并保留足够文字对比度
- 面板支持模糊背景层、细边框和轻微发光效果
- 深色主题下优先体现科技感，浅色主题下保留简洁与通透感
- 半透明视觉不能影响主要信息区可读性

#### US-03-02：卡片状态层级更清晰、更有质感

**作为** 需要快速浏览大量卡片的用户  
**我希望** 卡片在默认、悬停、选中、预览中状态下都有明显但克制的差异  
**以便** 我能快速感知当前焦点与可操作对象。

**验收要点**：
- 卡片默认态、hover 态、selected 态、previewing 态都有统一样式规则
- 选中态强化边框、阴影和高亮，不仅依赖背景色差异
- 卡片标题、正文、辅助信息层级更清晰，提升首屏扫读效率
- 图片 / 文件 / 文本三类卡片风格统一，但保留类型识别差异

## 5. 排除范围说明

- `v1.3` 的预览能力只在当前主面板内以覆盖层形式实现，不单独打开新窗口
- `v1.3` 的右键菜单优先承接当前已有能力，不扩展到“收藏”“标签”“分享”等新业务动作
- `v1.3` 的视觉升级以默认样式迭代为主，不提供“用户自定义主题编辑器”
- `v1.3` 不调整历史记录的数据库结构、排序规则与采集链路
