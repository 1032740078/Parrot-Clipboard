# v1.3 版本迭代日志记录

## 1. 迭代总览

`v1.3` 当前阶段先完成文档基线建设，锁定以下 3 个版本目标：

- 选中卡片后按空格，可在屏幕中心打开完整内容预览
- 卡片支持右键菜单，通过鼠标完成预览、粘贴、纯文本粘贴、删除等主要操作
- 主面板与卡片样式升级为半透明玻璃科技风，提升现代感与质感

## 2. 本次变更明细

### 2.15 2026-03-09 `v1.3-add-tauri-dev-script`

**变更类型**：`chore(dev-script)`

**主要变更**：

- 更新 `package.json`，新增 `tauri:dev` 脚本，在执行 `tauri dev` 前自动补上 `${CARGO_HOME:-$HOME/.cargo}/bin` 到 `PATH`

**产出结果**：

- 现在可以直接通过 `pnpm run tauri:dev` 启动桌面联调环境，即使 IDE / nvm 启动环境默认没带上 `cargo` 路径也能识别 Rust 工具链

## 3. 当前阶段结论

### 已锁定事项

- `v1.3` 的目标是补齐“完整内容确认”和“纯鼠标操作”两条体验链路
- 空格聚焦预览与右键菜单都优先复用既有详情、粘贴、删除命令，不扩张后端命令面
- 半透明玻璃科技风是 `v1.3` 的默认视觉方向，而不是用户可配置选项
- `v1.3` 仍不新增数据库表、不升级配置结构

### 下一步建议

1. 基于当前文档开始拆分 `PreviewOverlay`、`CardContextMenu` 与样式令牌实现任务
2. 优先完成空格预览与右键菜单的运行态设计，再进入视觉细化
3. 在实现阶段尽早引入视觉回归测试，防止科技风样式在不同主题下失真
