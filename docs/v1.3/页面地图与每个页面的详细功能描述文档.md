## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 功能定位 | 高频浏览、选择、预览、粘贴和删除剪贴板记录 |
| 打开方式 | 全局快捷键、托盘菜单、窗口显隐事件 |
| 关闭方式 | `Esc`、失焦、粘贴成功后自动隐藏 |
| 核心变化 | 半透明科技风、空格预览、右键菜单 |

### 3.2 `v1.3` 变化点

- 面板背景从普通纯色浮层升级为半透明玻璃质感
- 卡片支持右键菜单，补齐鼠标主导的操作入口
- 选中卡片支持空格打开完整内容预览层

### 3.3 区域说明

| 区域 | 说明 |
|------|------|
| 顶部标题区 | 显示应用标题、运行状态与辅助操作 |
| 卡片列表区 | 文本 / 图片 / 文件卡片横向或虚拟列表 |
| 右上角快捷徽标区 | 延续 `1~9` 可视编号逻辑 |
| 底部快捷提示区 | 提示 `空格预览`、`Enter 粘贴`、`Delete 删除` 等操作 |
| 浮层区 | 承载右键菜单、Toast、预览层遮罩 |

### 3.4 状态变体

| 状态 | 说明 |
|------|------|
| 默认态 | 面板可见，存在默认玻璃背景 |
| Hover 态 | 鼠标移动到某卡片上，卡片有轻微高亮和阴影 |
| Selected 态 | 当前选中卡片拥有明确边框 / 光晕 / 强化层次 |
| Previewing 态 | 主面板仍可见，但视觉焦点被预览层接管 |
| Context Menu Open 态 | 某张卡片上方存在右键菜单 |

### 3.5 关键交互规格

- 左右键、数字键、`Enter`、`Shift + Enter` 保持 `v1.2` 行为
- `Space`：打开 / 关闭当前选中卡片的聚焦预览
- `右键`：先选中当前卡片，再弹出右键菜单
- 点击菜单项后，动作结果与既有快捷键语义一致

## 4. P01-1 聚焦预览层（Preview Overlay）

### 4.1 功能定位

- 用于在不离开主面板的情况下查看完整内容
- 属于 `v1.3` 新增覆盖层视图，不单独出现在窗口列表中

### 4.2 内容组成

| 区域 | 说明 |
|------|------|
| 预览头部 | 类型图标、标题、时间、辅助操作提示 |
| 预览主体 | 完整文本 / 大图 / 文件清单 |
| 预览底部 | 关闭提示、可选快捷提示 |
| 背景遮罩 | 弱化主面板，突出当前预览内容 |

### 4.3 状态变体

| 状态 | 说明 |
|------|------|
| Loading | 正在读取详情 |
| Ready | 完整内容已可见 |
| Error | 详情读取失败，显示局部错误信息 |
| Closing | 预览关闭动画中 |

### 4.4 行为规格

- 打开方式：`Space`、右键菜单中的 `预览完整内容`
- 关闭方式：`Esc`、再次按 `Space`、点击遮罩
- 关闭后保留主面板原选中项，不改变滚动位置

## 5. P02 设置窗口（Settings Window）

### 5.1 功能定位

- 提供设置项查看与配置能力
- `v1.3` 不新增设置页分组或字段

### 5.2 与 `v1.3` 的关系

- 设置页不承接右键菜单和预览能力
- 如未来开放视觉偏好设置，再考虑在该页新增 `外观` 分组

---

## 6. P03 关于页（About Window）

### 6.1 功能定位

- 应用信息、版本信息、更新状态查看

### 6.2 与 `v1.3` 的关系

- `v1.3` 不调整其功能与布局

---

## 7. P04 权限引导弹窗（Permission Guide）

### 7.1 触发条件

- 粘贴相关动作因权限或平台能力受限而失败时触发

### 7.2 行为规格

- 菜单粘贴失败时也沿用同一套权限引导逻辑
- 权限弹窗优先级高于右键菜单，但不应错误覆盖已打开的预览层内容

---

## 8. P05 托盘菜单（Tray Menu）

### 8.1 菜单项说明

- 打开主面板
- 打开设置窗口
- 打开关于页
- 退出应用

### 8.2 与 `v1.3` 的关系

- `v1.3` 不调整托盘菜单结构
- 托盘菜单仍是无法使用快捷键时的主入口之一

---

## 9. 视图组件说明

### 9.1 `V31` 卡片上下文菜单

- 定位：鼠标右键后的局部悬浮菜单
- 功能：预览、直接粘贴、纯文本粘贴、删除
- 特性：危险动作独立分组、禁用项可见但不可触发

### 9.2 `V32` 聚焦预览卡片

- 定位：居中放大卡片容器
- 功能：完整显示记录内容，并保留类型识别样式
- 特性：支持文本滚动、图片大图显示、文件列表展示

### 9.3 `V33` 玻璃面板外壳

- 定位：主面板统一背景与层级容器
- 功能：提供半透明、模糊、边框、光效
- 特性：深浅主题共享一套设计语言但参数不同

---

## 10. 页面间状态转换

- `P01 -> P01-1`：用户按空格或点击菜单预览
- `P01-1 -> P01`：用户按 `Esc`、空格或点击遮罩关闭
- `P01 -> V31`：用户右键卡片
- `V31 -> P01-1`：点击 `预览完整内容`
- `V31 -> 隐藏主面板`：点击 `直接粘贴` 或 `纯文本粘贴` 且成功
- `V31 -> P01`：点击 `删除记录`、点击外部区域或按 `Esc`

---

## 11. 可访问性要求

- 菜单项最小可点击区域满足桌面指针操作要求
- 预览层和菜单层应具备清晰焦点与层级语义
- 半透明背景下仍需保证文本对比度与信息可辨识度
- 动效应保持轻量，必要时支持系统“减少动态效果”偏好
