## 3. 当前版本结论

### 已锁定事项

- 文本、图片、文件三类记录共享主表，类型特有数据分表存储
- 图片原图与缩略图采用“数据库元数据 + 文件系统资源”的组合模式
- 主面板仍为唯一核心界面，不引入设置页、托盘与搜索页
- 纯文本粘贴仅对文本记录开放

### 下一步建议

1. 优先实现 SQLite 迁移与仓储层
2. 紧接着实现图片/文件监听与资源保存
3. 然后扩展粘贴链路与主面板 UI
4. 最后补齐自动化测试与 E2E 验收
