# v0.2 页面地图与每个页面的详细功能描述文档

## 1. 文档概述

### 1.1 版本范围

`v0.2` 仍然只有一个核心界面：**主面板（Main Panel）**。当前版本不包含设置页、托盘菜单与搜索页。

### 1.2 v0.2 界面清单

| 编号 | 名称 | 类型 | 说明 |
|------|------|------|------|
| P01 | 主面板 | 页面 | 历史记录浏览与操作入口 |
| V01 | 文本卡片 | 组件视图 | 文本摘要展示 |
| V02 | 图片卡片 | 组件视图 | 图片缩略图与尺寸展示 |
| V03 | 文件卡片 | 组件视图 | 文件集合摘要展示 |
| V04 | 空状态视图 | 组件视图 | 无历史时引导用户先复制内容 |
| V05 | 骨架卡片 | 组件视图 | 初次加载或缩略图未就绪时使用 |

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 页面类型 | 悬浮主窗口 |
| 触发方式 | 全局快捷键 `Shift + Command + V` |
| 关闭方式 | `Esc`、点击外部区域、成功粘贴后自动关闭 |
| 主要目标 | 浏览最近记录并快速再次粘贴 |

### 3.2 功能定位

- 展示最近历史记录的混合卡片列表
- 支持键盘快速选择、删除、粘贴
- 在不打开复杂设置的前提下完成 90% 的高频使用场景

### 3.3 布局结构

```text
┌────────────────────────────────────────────────────────────────────┐
│ 粘贴板历史 · v0.2 Alpha                     最近记录 / 已监听      │
├────────────────────────────────────────────────────────────────────┤
│  ┌────────────┐  ┌────────────┐  ┌────────────┐  ┌────────────┐    │
│  │ V01 文本卡片 │  │ V02 图片卡片 │  │ V03 文件卡片 │  │ V02 图片卡片 │    │
│  └────────────┘  └────────────┘  └────────────┘  └────────────┘    │
├────────────────────────────────────────────────────────────────────┤
│ Enter 粘贴   Shift+Enter 纯文本   Delete 删除   Esc 关闭           │
└────────────────────────────────────────────────────────────────────┘
```

### 3.4 区域说明

| 区域 | 功能 |
|------|------|
| 顶部标题区 | 显示产品名、版本标识、监听状态 |
| 卡片列表区 | 横向展示混合记录卡片，最新在左 |
| 底部提示区 | 展示当前可用快捷键与模式说明 |

### 3.5 键盘交互规格

| 按键 | 行为 |
|------|------|
| `←` / `→` | 在卡片之间切换选中状态 |
| `Enter` | 原格式粘贴当前选中记录 |
| `Shift + Enter` | 对文本记录执行纯文本粘贴 |
| `Delete` / `Backspace` | 删除当前选中记录 |
| `Esc` | 关闭面板 |

### 3.6 状态变体

| 状态 | 说明 |
|------|------|
| 正常态 | 有记录，卡片列表可用 |
| 首次加载态 | 读取 SQLite 摘要期间展示 `V05` 骨架卡片 |
| 空状态 | 无任何历史记录时展示 `V04` |
| 缩略图占位态 | 图片卡片缩略图未就绪时展示占位图和加载骨架 |
| 错误提示态 | 底部 Toast 显示粘贴失败或资源不可用提示 |

### 3.7 行为规格（边界情况）

- 删除最后一条记录后，主面板立即切换到空状态
- 当前选中图片卡片在缩略图刷新时，不应丢失焦点态
- 文件记录被删除时，仅删除历史记录，不删除用户原始文件
- 对非文本卡片按 `Shift + Enter` 时，前端应禁用该操作并保持面板打开

## 4. V01 文本卡片（Text Card）

### 4.1 展示信息

| 信息项 | 说明 |
|--------|------|
| 标题 | 固定显示“文本”或来源应用名 |
| 时间 | 显示相对时间，如“刚刚”“2 分钟前” |
| 预览 | 最多显示前 3 行文本，超出省略 |
| 元信息 | 字符数、行数 |

### 4.2 交互状态

| 状态 | 视觉表现 |
|------|----------|
| 默认态 | 中性背景 |
| 悬停态 | 阴影略增强 |
| 选中态 | 高亮边框与更深背景 |

---

## 5. V02 图片卡片（Image Card）

### 5.1 展示信息

| 信息项 | 说明 |
|--------|------|
| 缩略图区 | 200×200 预览区域，保持可识别主体 |
| 标题 | 优先使用 `preview_text`，如截图名称 |
| 元信息 | `PNG/JPEG`、像素尺寸、文件大小摘要 |
| 状态标识 | `pending` / `failed` 使用不同占位提示 |

### 5.2 状态变体

| 状态 | 说明 |
|------|------|
| `pending` | 显示骨架块与“正在生成预览” |
| `ready` | 显示真实缩略图 |
| `failed` | 显示图片图标占位与“预览不可用” |

---

## 6. V03 文件卡片（File Card）

### 6.1 展示信息

| 信息项 | 说明 |
|--------|------|
| 图标 | 根据首个文件扩展名或目录类型展示 |
| 标题 | 首个文件名 |
| 数量摘要 | 如“共 4 项” |
| 附加标识 | 若包含目录，显示“含文件夹” |

### 6.2 行为说明

- 单文件卡片尽量突出文件名
- 多文件卡片突出“首项 + 数量”结构
- 文件路径不在卡片上完整展示，避免信息过载

---

## 7. V04 空状态视图（Empty State View）

### 7.1 展示目标

- 告知用户应用正在监听粘贴板
- 引导用户复制文本、图片或文件开始使用
- 保持与主面板整体视觉风格一致

### 7.2 文案建议

- 标题：`还没有历史记录`
- 副文案：`复制文本、图片或文件后，会自动出现在这里`
- 提示文案：`支持 Enter 粘贴，Shift+Enter 纯文本粘贴`

---

## 8. V05 骨架卡片（Skeleton Card）

### 8.1 使用场景

- 主面板首次从数据库加载摘要时
- 图片卡片缩略图未就绪时

### 8.2 交互要求

- 骨架态不阻塞方向键移动
- 骨架态切换为真实卡片时不闪烁、不跳动

---

## 9. 页面间状态转换

### 9.1 主面板状态机

```text
隐藏
  │ 快捷键唤起
  ▼
加载态
  ├── 有记录 → 正常态
  └── 无记录 → 空状态

正常态
  ├── Enter/Shift+Enter 成功 → 隐藏
  ├── Delete 删除最后一条 → 空状态
  ├── Delete 删除非最后一条 → 正常态
  └── Esc / 点击外部 → 隐藏
```

---

## 10. 可访问性要求

- 卡片列表必须有清晰的焦点态
- 纯文本粘贴不可用时应有可感知的禁用反馈
- 图片失败占位态需要文字说明，不能只依赖图标
