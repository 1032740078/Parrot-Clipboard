# 粘贴板记录管理工具 - 页面地图与页面功能简介

## 1. 文档概述

### 1.1 目的

本文档梳理应用内所有可见界面（页面、视图、弹窗、菜单）的完整地图，说明每个界面的功能定位、触发方式、包含的 UI 元素及交互逻辑，作为前端开发和 UI/UX 设计的统一参考。

### 1.2 界面分类说明

本应用是一个常驻后台的桌面工具，界面形态与传统多页面应用不同，分为以下几类：

| 类型 | 说明 |
|------|------|
| **面板（Panel）** | 快捷键触发、悬浮于屏幕底部的主操作界面 |
| **窗口（Window）** | 独立弹出的系统原生窗口，如设置窗口 |
| **菜单（Menu）** | 系统级右键菜单，如托盘菜单 |
| **引导页（Guide）** | 首次启动时的向导式界面 |
| **覆盖层（Overlay）** | 在主面板之上叠加的交互层，如搜索激活态 |

### P01 主面板

#### 基本信息

| 项目 | 说明 |
|------|------|
| 界面类型 | 面板（Panel） |
| 触发方式 | 全局快捷键（默认 `Shift + Cmd + V` / `Shift + Ctrl + V`） |
| 关闭方式 | 按 `Esc`、点击面板外区域、粘贴操作完成后自动关闭 |
| 位置 | 屏幕底部，宽度占满屏幕，高度自适应（约 180-300px） |
| 视觉风格 | 无边框、半透明毛玻璃背景、始终置顶 |
| 对应模块 | M05 主面板 UI 模块 |

#### 功能定位

主面板是用户的**核心操作界面**，承担 90% 的日常交互：查看历史记录、选择记录、执行粘贴、删除记录。设计目标是「**弹出即用，用完即走**」，最短路径内完成操作。

#### 布局结构

```
┌─────────────────────────────────────────────────────────────────────┐
│  [工具栏]  🔍  ⊙ 剪贴板  +                              ···        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐  │
│  │ 文本卡片  │ │ 文本卡片  │ │ 文件卡片  │ │ 图片卡片  │ │ 图片卡片  │  │
│  │ [选中态] │ │          │ │          │ │          │ │          │  │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘ └──────────┘  │
│  ◀ 左右滚动查看更多记录                                            ▶ │
└─────────────────────────────────────────────────────────────────────┘
```

#### 区域说明

**工具栏（Toolbar）**

| 元素 | 类型 | 功能 |
|------|------|------|
| 搜索图标 `🔍` | 按钮 | 点击激活搜索覆盖态（P01-S），面板进入搜索模式 |
| 类型标签 `⊙ 剪贴板` | 切换标签 | 按内容类型筛选（全部 / 文本 / 图片 / 文件）；图标为时钟表示"最近" |
| 新建按钮 `+` | 按钮 | 快速将当前粘贴板内容钉入记录（手动触发保存，保留置顶） |
| 更多菜单 `···` | 按钮 | 展开快捷操作（清空全部、暂停监听、打开设置） |

**卡片列表区（Card List）**

- 横向排列，按复制时间倒序（最新在最左）
- 支持鼠标横向滚动、键盘方向键切换
- 最多显示前 N 条记录（由配置决定，默认 200 条文本 / 50 条图片）
- 选中状态：当前聚焦卡片高亮显示边框和背景

#### 键盘快捷键

| 按键 | 动作 |
|------|------|
| `←` / `→` | 切换选中记录 |
| `Enter` | 以原格式粘贴选中记录，面板自动关闭 |
| `Shift + Enter` | 以纯文本模式粘贴，剥除所有富文本格式，面板自动关闭 |
| `Delete` / `Backspace` | 删除选中记录，自动聚焦下一条 |
| `Esc` | 关闭主面板，不执行任何操作 |
| `1` ~ `9` | 快速跳转并选中第 1 ~ 9 条记录 |
| `/` 或 `Cmd/Ctrl + F` | 激活搜索模式（P01-S） |

#### 状态变体

| 状态 | 说明 |
|------|------|
| 正常态 | 有记录，展示卡片列表 |
| 空状态（V04） | 无任何历史记录时显示空状态提示 |
| 搜索态（P01-S） | 搜索框展开，卡片列表过滤为匹配结果 |
| 监听暂停态 | 工具栏显示「已暂停」标识，提示用户当前不在记录 |
| 加载态 | 首次打开时，数据库读取期间显示骨架屏 |

### P01-S 搜索覆盖态（主面板搜索模式）

#### 基本信息

| 项目 | 说明 |
|------|------|
| 界面类型 | 覆盖层（主面板内部状态切换，非独立窗口） |
| 触发方式 | 点击工具栏搜索图标 / 按下 `/` 或 `Cmd+F` |
| 关闭方式 | 按 `Esc`（退出搜索，返回正常态，清空搜索词） |
| 对应模块 | M05 主面板 UI 模块 |

#### 功能说明

工具栏搜索图标展开为全宽搜索输入框，卡片列表实时过滤（输入防抖 300ms）。搜索范围为文本记录的内容文字；图片和文件记录按来源应用名搜索。

#### 布局变化

```
┌─────────────────────────────────────────────────────────────────────┐
│  [工具栏]  🔍 [________________________搜索输入框___________] × Esc │
├─────────────────────────────────────────────────────────────────────┤
│  搜索结果：共 N 条匹配                                               │
│  ┌──────────┐ ┌──────────┐                                          │
│  │ 文本卡片  │ │ 文本卡片  │  （仅显示匹配记录）                       │
│  └──────────┘ └──────────┘                                          │
└─────────────────────────────────────────────────────────────────────┘
```

### V01 文本卡片

#### 基本信息

| 项目 | 说明 |
|------|------|
| 界面类型 | 卡片视图组件 |
| 归属页面 | P01 主面板卡片列表 |
| 对应模块 | M05 主面板 UI 模块 |

#### 布局结构（参考原型图）

```
┌──────────────────────────────┐
│ 文本        [来源应用图标]    │  ← 顶部标题栏（色块背景，与内容类型对应）
├──────────────────────────────┤
│ N 小时前                     │  ← 相对时间戳
├──────────────────────────────┤
│ 文本内容预览（前 3 行）        │
│ 超长内容末尾省略...           │
│                              │
│            🔒               │  ← 锁图标（来自黑名单应用时模糊/锁定）
└──────────────────────────────┤
│ N 个字符        ≡ 序号        │  ← 底部元信息栏
└──────────────────────────────┘
```

#### 展示信息

| 信息项 | 说明 |
|--------|------|
| 顶部色块 | 内容类型色标（文本：蓝/橙等，由来源应用主色决定）|
| 来源应用图标 | 显示复制来源的应用图标（如 Chrome、微信）|
| 相对时间 | 如「6 小时前」「刚刚」「昨天」 |
| 文本预览 | 最多显示前 3 行，单行最多 20 字，超出截断 |
| 字符数 | 底部显示总字符数 |
| 序号标记 | 底部右侧显示 `≡ N`，对应键盘数字键快捷选取 |

#### 交互状态

| 状态 | 视觉变化 |
|------|----------|
| 默认态 | 正常展示 |
| 悬停态 | 轻微阴影增强，背景轻微高亮 |
| 选中态 | 高亮边框（主题色），背景色加深 |
| 黑名单应用 | 文本内容区模糊处理 + 锁图标覆盖 |

### P02-2 记录与存储页

#### 功能定位

管理历史记录的保存策略和存储空间使用，帮助用户在功能与性能之间取得平衡。

#### 配置项清单

| 配置项 | 控件类型 | 默认值 | 说明 |
|--------|----------|--------|------|
| 文本记录保留数量 | 数字步进器 / 滑块 | 200 条 | 范围 10 ~ 1000 |
| 图片记录保留数量 | 数字步进器 / 滑块 | 50 条 | 范围 5 ~ 200 |
| 是否记录图片 | 开关 Toggle | 开启 | 关闭后不再捕获图片类型 |
| 单张图片大小上限 | 数字输入 + 单位选择 | 10 MB | 超出时询问是否保存 |
| 总存储空间上限 | 数字输入 + 单位选择 | 500 MB | 超出后自动清理最旧记录 |
| 当前已用空间 | 只读展示 | — | 显示「已用 XX MB / XX MB」进度条 |
| 立即清理冗余文件 | 按钮 | — | 清理孤立图片文件（不删除记录） |

### P04 托盘右键菜单

#### 基本信息

| 项目 | 说明 |
|------|------|
| 界面类型 | 系统菜单（Menu） |
| 触发方式 | 右键点击菜单栏（macOS）/ 系统托盘（Windows/Linux）图标 |
| 对应模块 | M11 系统托盘模块 |

#### 菜单项定义

```
┌──────────────────────────┐
│  显示面板 / 隐藏面板       │  ← 根据当前面板状态动态切换文字
├──────────────────────────┤
│  ──────────（分隔线）      │
├──────────────────────────┤
│  ⏸ 暂停监听              │  ← 监听中时显示「暂停」，暂停中显示「恢复监听」
├──────────────────────────┤
│  ──────────              │
├──────────────────────────┤
│  🗑 清空历史记录           │  ← 点击触发 P05 确认弹窗
│  ⚙ 设置                  │  ← 打开 P02 设置窗口
├──────────────────────────┤
│  ──────────              │
├──────────────────────────┤
│  ℹ 关于                  │  ← 直接打开 P02-5 关于页
│  ✕ 退出                  │
└──────────────────────────┘
```

#### 托盘图标状态

| 状态 | 图标变化 | 说明 |
|------|----------|------|
| 正常监听中 | 标准图标（实心） | 应用正常运行 |
| 监听已暂停 | 图标变灰 / 带暂停标记 | 提示用户当前不在记录 |
| macOS 深色模式 | 使用 Template Image 自适应 | 系统自动适配白/黑色 |

### P05 通用确认弹窗

#### 基本信息

| 项目 | 说明 |
|------|------|
| 界面类型 | 模态弹窗（Dialog） |
| 触发场景 | 清空全部记录、删除黑名单条目等不可逆操作前 |

#### 弹窗内容

```
┌─────────────────────────────────┐
│  确认清空全部记录？               │
│                                 │
│  此操作将删除所有 N 条历史记录，   │
│  且无法恢复。                    │
│                                 │
│        [取消]    [确认清空]       │
└─────────────────────────────────┘
```

- 取消按钮：次要样式，关闭弹窗，不执行任何操作
- 确认按钮：危险样式（红色），执行对应操作
