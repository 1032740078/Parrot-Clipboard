## 4. 风险与缓解

| 风险 | 说明 | 缓解方案 |
|------|------|----------|
| 托盘 API 平台差异 | Tauri 托盘在不同 macOS 版本表现可能不同 | 先锁定 macOS 单平台，增加 Mock 与真机回归 |
| Launch Agent 写入权限 | 用户环境可能限制写入或存在旧残留 | 统一走临时目录测试，真实环境错误写日志并回滚 UI |
| 多显示器定位复杂 | Dock 位置与工作区计算容易出错 | 独立 `window/position.rs` 并补齐布局夹具 |

---

## 5. 冲刺完成后建议

- 完成 `v0.3` 后再进入 `v0.5` 设置页，避免在无系统集成基线时直接堆配置 UI
- 若托盘与自启动落地稳定，可在 `v0.5` 复用当前系统状态接口给设置页
