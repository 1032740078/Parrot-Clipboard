# v0.3 页面地图与每个页面的详细功能描述文档

## 1. 文档概述

### 1.1 版本范围

`v0.3` 仅关注以下界面与交互载体：
- 主面板（Main Panel）
- 系统托盘菜单（Tray Menu）
- 清空历史确认弹窗（Confirm Dialog）

当前版本不包含设置页、搜索页、收藏页。

### 1.2 v0.3 界面清单

| 编号 | 名称 | 类型 |
|------|------|------|
| P01 | 主面板 | 面板 |
| P04 | 托盘菜单 | 系统菜单 |
| P05 | 清空历史确认弹窗 | 弹窗 |
| V01 | 文本卡片 | 视图组件 |
| V02 | 图片卡片 | 视图组件 |
| V03 | 文件卡片 | 视图组件 |
| V04 | 空状态视图 | 视图组件 |
| V05 | 骨架卡片 | 视图组件 |
| V06 | 监听暂停提示条 | 视图组件 |

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 触发方式 | 全局快捷键、托盘菜单 |
| 关闭方式 | `Esc`、点击托盘隐藏、粘贴后自动关闭 |
| 位置 | 当前活动显示器底部 `work_area` |
| 视觉风格 | 无边框、半透明、始终置顶 |

### 3.2 功能定位

主面板仍然是 `v0.3` 的核心操作界面，负责：
- 浏览历史记录
- 通过方向键或数字键快速选中
- 执行粘贴 / 纯文本粘贴 / 删除
- 在监听暂停时展示弱提示，但仍允许浏览历史

### 3.3 布局结构

```text
┌─────────────────────────────────────────────────────────────────────┐
│ [V06 监听已暂停，可从托盘恢复]                               ×     │
├─────────────────────────────────────────────────────────────────────┤
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐               │
│  │ 文本卡片  │ │ 图片卡片  │ │ 文件卡片  │ │ 文本卡片  │               │
│  │ [选中态] │ │          │ │          │ │          │               │
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘               │
├─────────────────────────────────────────────────────────────────────┤
│ Enter 粘贴  Shift+Enter 纯文本  Delete 删除  1-9 快选  Esc 关闭    │
└─────────────────────────────────────────────────────────────────────┘
```

### 3.4 区域说明

| 区域 | 说明 |
|------|------|
| 顶部提示区 | 仅在监听暂停时显示 `V06` |
| 卡片列表区 | 横向滚动、支持选中高亮 |
| 底部快捷键栏 | 展示 Enter / Shift+Enter / Delete / `1-9` / Esc |

### 3.5 键盘交互规格

| 按键 | 动作 |
|------|------|
| `←` / `→` | 切换选中记录 |
| `1-9` | 快速选中第 1~9 条记录 |
| `Enter` | 按原格式粘贴当前记录 |
| `Shift + Enter` | 文本记录按纯文本模式粘贴 |
| `Delete` / `Backspace` | 删除当前记录 |
| `Esc` | 关闭主面板 |

### 3.6 状态变体

| 状态 | 说明 |
|------|------|
| 正常态 | 有记录且监听正常 |
| 监听暂停态 | 顶部显示弱提示，不阻断浏览 |
| 空状态 | 无记录时显示 `V04` |
| 加载态 | 首次读取摘要时显示 `V05` |

### 3.7 行为规格（边界情况）

- 记录不足 9 条时，超出的数字键无动作
- 删除最后一条记录后直接切换为空状态
- 监听暂停时仍可粘贴历史记录，但不会记录新的复制内容
- 从托盘打开主面板时，与快捷键打开保持相同的布局与焦点策略

## 4. P04 托盘菜单（Tray Menu）

### 4.1 基本信息

| 项目 | 说明 |
|------|------|
| 类型 | 系统级右键菜单 |
| 触发方式 | 点击菜单栏托盘图标 |
| 对应模块 | `tray/` |

### 4.2 菜单项说明

| 菜单项 | 类型 | 说明 |
|--------|------|------|
| 显示主面板 / 隐藏主面板 | 普通项 | 根据当前窗口状态动态切换文案 |
| 暂停监听 / 恢复监听 | 普通项 | 根据当前监听状态动态切换文案 |
| 开机自启动 | 勾选项 | 显示当前 `launch_at_login` |
| 清空历史 | 危险项 | 触发 `P05` 确认弹窗 |
| 打开日志目录 | 普通项 | 便于排查问题 |
| 退出应用 | 危险项 | 退出整个应用 |

### 4.3 状态反馈

- 监听正常：托盘图标正常态
- 监听暂停：托盘图标弱化或暂停态
- 托盘初始化失败：记录日志，主流程改走快捷键，不展示菜单

---

## 5. P05 清空历史确认弹窗（Confirm Dialog）

### 5.1 基本信息

| 项目 | 说明 |
|------|------|
| 触发方式 | 托盘点击“清空历史” |
| 关闭方式 | 取消、确认后自动关闭 |
| 风险等级 | 高风险操作确认 |

### 5.2 展示目标

- 明确提醒会删除文本、图片、文件三类历史
- 明确提醒图片预览资源也会被删除
- 提供二次确认，降低误触概率

### 5.3 文案建议

- 标题：`确认清空全部历史？`
- 描述：`这将删除所有文本、图片、文件记录及其预览资源，且不可撤销。`
- 主按钮：`确认清空`
- 次按钮：`取消`

## 6. V01 文本卡片（Text Card）

### 6.1 展示信息

- 文本预览
- 来源应用
- 相对时间
- 字符数
- 数字键槽位标记（前 9 条可显示 `1-9`）

### 6.2 交互状态

- 默认态
- 悬停态
- 选中态
- 删除离场态

---

## 7. V02 图片卡片（Image Card）

### 7.1 展示信息

- 缩略图或占位图
- 图片尺寸
- MIME 摘要
- 来源应用与时间

### 7.2 状态变体

- `pending`
- `ready`
- `failed`
- 选中态 / 删除离场态

---

## 8. V03 文件卡片（File Card）

### 8.1 展示信息

- 首个文件名
- `+N` 数量摘要
- 是否包含目录
- 来源应用与时间

### 8.2 行为说明

- 单文件显示文件名
- 多文件显示首项名称 + 数量
- 含目录显示目录标识，避免误判为单文件

---

## 9. V04 / V05 / V06 辅助视图

### 9.1 V04 空状态视图

- 用于“清空历史后”与“首次无记录”两种场景
- 建议文案：`还没有历史记录，先去复制一些内容吧`

### 9.2 V05 骨架卡片

- 用于主面板首次加载摘要时
- 数量建议为 `3`

### 9.3 V06 监听暂停提示条

- 文案建议：`监听已暂停，新复制的内容不会被记录`
- 视觉应弱于错误 Toast，避免制造紧张感

---

## 10. 页面间状态转换

### 10.1 状态机

```text
隐藏态
  ├── 快捷键 / 托盘显示 -> 主面板可见态
  └── 托盘退出 -> 应用退出

主面板可见态
  ├── Esc / 粘贴完成 / 托盘隐藏 -> 隐藏态
  ├── 托盘暂停监听 -> 主面板可见 + 监听暂停提示态
  └── 清空历史确认通过 -> 空状态
```

## 11. 可访问性要求

- `1-9` 快选提示必须在 UI 中可见，不能只存在于文档中
- `prefers-reduced-motion` 时动效降级
- 确认弹窗按钮焦点默认落在“取消”，降低误清空风险
