# v1.5 页面地图与每个页面的详细功能描述文档

## 1. 文档概述

### 1.1 版本范围

本文档仅描述 `v1.5` 当前版本的页面地图与页面 / 视图功能说明，聚焦音效反馈、预览自动联动、音频 / 视频 / 文稿 / 链接预览。

### 1.2 `v1.5` 界面清单

| 页面 / 视图 | 是否有变化 | 说明 |
|-------------|------------|------|
| `P01` 主面板（Main Panel） | 是 | 新增预览联动触发与音效反馈结果承载 |
| `P02` 预览窗口（Preview Window） | 是 | 本版本核心变化承载页 |
| `P03` 设置窗口（Settings Window） | 否 | 本版不新增配置入口 |
| `P04` 关于窗口（About Window） | 否 | 本版不新增变化 |
| `P05` 权限引导窗口（Permission Guide） | 否 | 沿用既有逻辑 |
| `P06` 托盘菜单（Tray Menu） | 否 | 继续作为主面板打开入口 |

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 功能定位 | 浏览、搜索、筛选、选中、打开预览、执行粘贴 |
| 打开方式 | 全局快捷键、托盘菜单、窗口显隐事件 |
| 关闭方式 | `Esc`、失焦、粘贴成功后自动隐藏 |
| 核心变化 | 当预览窗口可见时，选中变化可驱动预览联动 |

### 3.2 `v1.5` 变化点

- 预览窗口可见时，切换当前选中卡片会自动同步预览目标
- 复制成功、粘贴成功、预览显示成功后会触发对应音效反馈
- 主面板本身不新增新的大型 UI 区块，仍保持 `v1.4` 的搜索与筛选结构

### 3.3 区域说明

| 区域 | 说明 |
|------|------|
| 左侧筛选栏 | 沿用 `全部 / 文本 / 图片 / 文件 / 超链接 / 视频 / 音频 / 文稿` |
| 顶部搜索区 | 沿用动态宽度搜索框 |
| 卡片列表区 | 继续作为选中态与预览联动的主要触发区 |
| 底部快捷提示区 | 沿用 `v1.4` 布局 |

### 3.4 状态变体

| 状态 | 说明 |
|------|------|
| 默认态 | 浏览历史记录 |
| 预览联动态 | 预览窗口可见，主面板选中变化会同步驱动预览 |
| 粘贴执行态 | 粘贴完成后隐藏面板并触发音效 |
| 错误降级态 | 当前记录预览资源异常，但主面板仍可继续切换其他记录 |

### 3.5 关键交互规格

- 显式打开预览后，主面板进入“联动允许”状态
- 预览窗口关闭后，主面板恢复为普通浏览模式
- 快速切换卡片时只要求最终停留的预览结果正确，不要求每次中间态都完整渲染

## 4. P02 预览窗口（Preview Window）

### 4.1 功能定位

- 用于查看当前记录的完整内容
- 在 `v1.5` 中升级为可承载 `text / image / audio / video / pdf / document / link / file_list` 的统一预览工作区
- 支持在窗口已打开时自动切换预览目标记录

### 4.2 内容组成

| 区域 | 说明 |
|------|------|
| 头部标题区 | 记录标题、类型、来源应用、关闭操作 |
| 渲染器工作区 | 根据 `preview_renderer` 显示具体内容 |
| 降级提示区 | 资源准备中、失败、不支持时的统一反馈区域 |
| 辅助信息区 | 文件名、URL、时长、页数、站点信息等元数据 |

### 4.3 状态变体

| 状态 | 说明 |
|------|------|
| Loading | 正在加载详情或等待预览资源准备 |
| Ready | 当前渲染器可直接显示内容 |
| Pending Asset | 资源仍在后台准备 |
| Fallback | 当前记录只能显示文件名、路径、URL 或错误说明 |
| Error | 预览读取或渲染失败 |

### 4.4 行为规格

- 首次成功显示内容时触发预览音效
- 预览窗口已打开时，切换目标记录不重建窗口
- 文本预览继续支持编辑与自动保存
- 图片预览继续支持缩放与拖拽
- 音频 / 视频预览支持原生播放器控件
- 文稿 / 链接预览在能力不足时进入结构化内容或基础信息降级态

### 7.1 触发条件

- 粘贴或系统控制因权限受限而失败

## 9. 视图组件说明

### 9.1 `V51` 预览渲染器路由层

- 定位：预览窗口主工作区入口
- 功能：根据 `preview_renderer` 选择文本、图片、音频、视频、文稿、链接或 fallback 视图
- 特性：统一处理 `pending / ready / failed / unsupported`

### 9.2 `V52` 音频预览组件

- 定位：预览窗口媒体渲染器之一
- 功能：播放、暂停、拖动音频进度
- 特性：显示时长、文件名、MIME 等基础信息

### 9.3 `V53` 视频预览组件

- 定位：预览窗口媒体渲染器之一
- 功能：播放、暂停、拖动视频进度
- 特性：支持首帧 / 封面图与降级提示

### 9.4 `V54` 文稿预览组件

- 定位：预览窗口文稿渲染器之一
- 功能：显示 PDF 正文或结构化 Office 内容
- 特性：支持 loading、失败与旧格式降级

### 9.5 `V55` 链接预览组件

- 定位：预览窗口链接渲染器之一
- 功能：显示 URL、标题、摘要、站点信息与外部打开入口
- 特性：不依赖完整网页 iframe 才能成立

---

## 10. 页面间状态转换

- `P01 -> P02`：用户显式打开记录预览
- `P01(选中变化) -> P02`：预览窗口已打开时，当前预览目标切换
- `P02 -> P01`：用户关闭预览窗口，联动会话结束
- `P01 -> P05`：粘贴权限受限触发权限引导
- `P01 / P02 -> 系统反馈`：复制、粘贴、显示预览成功时触发音效反馈

---

## 11. 可访问性要求

- 音频 / 视频控件必须可键盘操作
- 文稿和链接降级态不能只靠颜色表达，必须有文字说明
- 预览加载中、失败、不支持三种状态必须有明确可读文案
- 关闭预览后焦点应回到主面板列表或稳定可预期位置
