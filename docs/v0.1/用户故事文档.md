# v0.1 用户故事文档

## 1. 文档概述

### 1.1 版本范围

本文档聚焦于 **v0.1 MVP 版本**的用户故事，覆盖内容：

- 纯文本粘贴板监听与记录
- 全局快捷键调出/隐藏主面板
- 横向卡片列表展示最近 20 条文本记录
- 键盘操作（方向键切换、Enter 粘贴、Delete 删除、Esc 关闭）
- 内存存储（不持久化，重启后清空）
- 单平台：macOS

### 1.2 角色定义

| 角色 | 说明 |
|------|------|
| **日常用户** | 频繁在多个应用间复制粘贴内容的 macOS 用户，追求高效工作流 |

## 3. 用户故事详细说明

---

### E01 - 粘贴板自动监听

#### US-01-01：自动记录文本复制内容

**作为**日常用户，
**我希望**应用在后台自动捕获我在任意应用中复制的文本，
**以便**我不需要手动保存就能随时找回历史复制内容。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-01-01-1 | 用户在任意 macOS 应用中执行复制（Cmd+C）后，应用在 500ms 内捕获该文本内容 |
| AC-01-01-2 | 捕获的内容被记录为新的历史条目，排在列表最左侧（最新位置） |
| AC-01-01-3 | 仅捕获纯文本内容，图片和文件路径在 v0.1 中不记录 |
| AC-01-01-4 | 连续快速复制时，每次复制内容均被独立记录（不合并） |
| AC-01-01-5 | 应用不需要用户手动干预，无后台 UI 弹出 |

**技术备注**：macOS 端通过轮询 `NSPasteboard.changeCount`（间隔 200ms）检测变化。

---

#### US-01-02：内容去重

**作为**日常用户，
**我希望**当我多次复制完全相同的文本时，历史列表不会出现重复条目，
**以便**历史列表保持简洁可用。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-01-02-1 | 当新复制的文本与列表最新一条记录内容完全相同时，不新增记录 |
| AC-01-02-2 | 若新内容与非最新记录相同（但不是最新），仍正常新增记录 |
| AC-01-02-3 | 去重比较区分大小写 |

---

#### US-01-03：记录数量上限控制

**作为**日常用户，
**我希望**历史记录最多保留 20 条，
**以便**列表不会无限增长导致面板过于拥挤。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-01-03-1 | 历史记录列表最多保留 20 条文本记录 |
| AC-01-03-2 | 当第 21 条记录被捕获时，最旧的一条（最右侧）记录被自动移除 |
| AC-01-03-3 | 自动移除操作对用户无感知，无任何提示弹出 |

---

### E02 - 快捷键调出面板

#### US-02-01：默认快捷键调出主面板

**作为**日常用户，
**我希望**按下 `Shift + Command + V` 后，粘贴板历史面板立即出现在屏幕底部，
**以便**我能快速查看和使用历史记录，无需切换任何应用。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-02-01-1 | 在任意 macOS 应用中按下 `Shift + Command + V`，主面板在屏幕底部弹出 |
| AC-02-01-2 | 从按键到面板可见的响应时间 < 100ms |
| AC-02-01-3 | 面板弹出时，最新的一条记录默认处于选中（高亮）状态 |
| AC-02-01-4 | 若当前面板已可见，再次按下快捷键则收起面板 |
| AC-02-01-5 | 面板弹出时自动获得键盘焦点 |

---

#### US-02-02：面板位于屏幕底部

**作为**日常用户，
**我希望**面板始终出现在屏幕最底部并横跨全屏宽度，
**以便**不遮挡当前工作内容，且位置固定易于预期。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-02-02-1 | 面板宽度等于当前屏幕宽度（100vw） |
| AC-02-02-2 | 面板位于屏幕底部，不超出屏幕边界 |
| AC-02-02-3 | 面板始终置顶，覆盖其他应用窗口 |
| AC-02-02-4 | 面板无系统标题栏（无边框窗口） |

---

### E03 - 历史记录浏览

#### US-03-01：查看文本历史记录卡片

**作为**日常用户，
**我希望**在面板中以横向卡片的形式看到我的历史复制记录，
**以便**快速视觉扫描并找到我需要的内容。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-03-01-1 | 面板展示横向排列的文本卡片，最新记录在最左侧 |
| AC-03-01-2 | 每张卡片显示文本内容的前 3 行，超出部分用省略号截断 |
| AC-03-01-3 | 每张卡片显示相对时间戳（如「刚刚」「6 分钟前」）|
| AC-03-01-4 | 卡片宽度固定为 240px，高度固定为 180px |
| AC-03-01-5 | 面板最多展示 20 条记录 |

---

#### US-03-02：通过方向键切换选中记录

**作为**日常用户，
**我希望**使用键盘左右方向键在历史记录之间快速切换，
**以便**不使用鼠标就能高效浏览历史内容。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-03-02-1 | 按 `→` 键时，选中状态移动到下一条记录（向右），当前选中卡片高亮显示 |
| AC-03-02-2 | 按 `←` 键时，选中状态移动到上一条记录（向左） |
| AC-03-02-3 | 选中项切换的响应时间 < 50ms |
| AC-03-02-4 | 已到达最左端时按 `←`，选中状态不变（不循环） |
| AC-03-02-5 | 已到达最右端时按 `→`，选中状态不变（不循环） |
| AC-03-02-6 | 选中卡片有明显的视觉高亮（高亮边框 + 背景加深） |

---

#### US-03-03：显示空状态

**作为**日常用户，
**我希望**当没有历史记录时，面板显示友好的空状态提示，
**以便**我了解当前没有内容，并知道如何开始使用。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-03-03-1 | 当历史记录为空时，卡片区域显示空状态提示文案 |
| AC-03-03-2 | 空状态文案包含「还没有复制记录」及使用引导提示 |
| AC-03-03-3 | 空状态无错误图标，只有提示性插画或图标 |

---

### E04 - 粘贴操作

#### US-04-01：Enter 键粘贴选中记录

**作为**日常用户，
**我希望**选中一条记录后按 `Enter`，内容自动粘贴到我当前工作的应用中，
**以便**无需鼠标点击即可完成粘贴操作。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-04-01-1 | 按下 `Enter` 后，选中记录的文本内容被写入系统粘贴板 |
| AC-04-01-2 | 写入粘贴板后，面板自动隐藏 |
| AC-04-01-3 | 面板隐藏后，系统模拟执行 `Cmd+V`，将内容粘贴到之前活动的应用中 |
| AC-04-01-4 | 从按下 Enter 到内容出现在目标应用的总耗时 < 200ms |
| AC-04-01-5 | 粘贴操作不产生新的历史记录（监听在粘贴时临时暂停） |

---

### E05 - 记录删除

#### US-05-01：Delete 键删除选中记录

**作为**日常用户，
**我希望**按 `Delete` 或 `Backspace` 键删除当前选中的记录，
**以便**清理不需要保留的历史内容。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-05-01-1 | 按 `Delete` 或 `Backspace` 键后，选中记录从列表中移除 |
| AC-05-01-2 | 删除后，如果被删除记录右侧有记录，则右侧第一条成为新的选中项 |
| AC-05-01-3 | 如果被删除记录是最后一条，则左侧第一条成为新的选中项 |
| AC-05-01-4 | 如果删除后列表为空，显示空状态视图 |
| AC-05-01-5 | 删除操作仅从内存中移除，v0.1 无持久化，重启即清空 |
| AC-05-01-6 | 删除响应时间 < 50ms |

---

### E06 - 关闭面板

#### US-06-01：Esc 键关闭面板

**作为**日常用户，
**我希望**按 `Esc` 键关闭主面板并返回之前的应用，
**以便**在不需要粘贴时快速退出而不影响工作流。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-06-01-1 | 按 `Esc` 键后，面板收起隐藏 |
| AC-06-01-2 | 面板收起时间 < 300ms |
| AC-06-01-3 | 面板收起不执行粘贴操作 |
| AC-06-01-4 | 面板收起后，焦点回到之前活动的应用 |

---

#### US-06-02：点击面板外区域关闭面板

**作为**日常用户，
**我希望**点击面板以外的区域时面板自动关闭，
**以便**通过鼠标点击操作也能快速退出面板。

**验收标准（AC）**：

| 编号 | 标准描述 |
|------|----------|
| AC-06-02-1 | 用户点击面板区域之外的屏幕区域时，面板自动收起 |
| AC-06-02-2 | 收起行为与 Esc 键行为一致，不执行粘贴 |

## 5. 排除范围说明

以下功能**不在 v0.1 范围内**，在用户故事中不应出现：

- 图片和文件类型的记录与展示
- 数据持久化（SQLite 存储）
- 系统托盘图标与菜单
- 开机自启动
- 设置界面
- 纯文本模式粘贴（Shift + Enter）
- Windows / Linux 平台支持
- 数字键 1-9 快速选择
- 搜索功能
