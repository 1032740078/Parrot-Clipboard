# v0.1 文件目录结构设计文档

## 1. 文档概述

### 1.1 版本范围

本文档描述 **v0.1 MVP 版本**的项目文件目录结构，聚焦于 v0.1 实际需要创建的文件，排除 v0.2+ 才引入的模块和文件。

### 1.2 v0.1 涉及模块

根据迭代规划，v0.1 涉及以下模块：

| 模块 | 版本状态 | 说明 |
|------|----------|------|
| M09 配置管理 | 基础版（内存配置） | 暂不持久化 |
| M14 平台抽象 | macOS 版 | 仅 macOS 实现 |
| M13 IPC 通信 | 核心接口 | 查询/粘贴/删除接口 |
| M01 粘贴板监听 | 文本版 | 仅监听文本内容 |
| M03 窗口管理 | 基础版 | 屏幕底部显示/隐藏 |
| M04 快捷键管理 | 基础版 | 注册调出面板快捷键 |
| M06 粘贴执行 | 文本版 | 仅文本粘贴，原格式 |
| M12 前端状态管理 | 基础版 | 记录列表、选中状态 |
| M05 主面板 UI | 基础版 | 文本卡片列表、键盘操作 |

**v0.1 不包含**：M02（SQLite）、M07（图片处理）、M08（自启动）、M10（设置界面）、M11（系统托盘）

## 3. 前端目录结构（`src/`）

```
src/
├── components/
│   ├── MainPanel/                      # [M05] 主面板 UI 模块
│   │   ├── index.tsx                   # 主面板容器，Framer Motion 弹出/收起动画
│   │   ├── CardList.tsx                # 卡片横向列表容器（v0.1 暂不使用虚拟滚动）

│   │   └── EmptyState.tsx              # 空状态视图组件

├── api/                                # [M13] IPC 通信封装（前端侧）
│   ├── commands.ts                     # invoke() 调用封装（get_records/delete/paste/hide）
│   ├── events.ts                       # listen() 订阅封装（new-record/record-deleted）

│   ├── useKeyboard.ts                  # [M05] 面板内键盘事件（←/→/Enter/Delete/Esc）
│   └── useClipboardEvents.ts           # [M12] 监听后端事件，驱动 store 更新
├── types/
│   ├── clipboard.ts                    # ClipboardRecord、ContentType、PasteMode
│   └── error.ts                        # IpcError、ErrorCode
├── __mocks__/
│   └── @tauri-apps/
│       └── api/
│           ├── core.ts                 # Mock invoke/listen（供 Vitest 使用）
│           └── event.ts                # Mock listen（Tauri event API）
├── __tests__/
│   ├── setup.ts                        # Vitest 全局设置（cleanup、mock 初始化）
│   ├── fixtures/
│   │   └── clipboardRecords.ts         # 测试用标准夹具数据
│   ├── stores/
│   │   ├── useClipboardStore.test.ts
│   │   └── useUIStore.test.ts
│   ├── components/
│   │   ├── MainPanel.test.tsx          # 键盘交互、卡片渲染测试
│   │   ├── TextCard.test.tsx           # 卡片展示、选中状态测试
│   │   └── EmptyState.test.tsx         # 空状态展示测试
│   └── hooks/
│       ├── useKeyboard.test.ts
│       └── useClipboardEvents.test.ts
├── App.tsx                             # 根组件（v0.1 仅主面板，无路由）
└── main.tsx                            # 前端入口，挂载 React
```

### 3.1 关键文件职责说明（v0.1）

| 文件 | 模块 | 职责 |
|------|------|------|
| `components/MainPanel/index.tsx` | M05 | 面板弹出动画容器（Framer Motion），包裹 CardList |
| `components/MainPanel/CardList.tsx` | M05 | 横向排列文本卡片，v0.1 最多 20 条，不使用虚拟滚动 |
| `components/MainPanel/TextCard.tsx` | M05 | 单张文本卡片，展示文本预览/时间戳/字符数 |
| `components/MainPanel/EmptyState.tsx` | M05 | 无记录时的提示视图 |
| `stores/useClipboardStore.ts` | M12 | 唯一前端记录数据源（records/selectedIndex） |
| `stores/useUIStore.ts` | M12 | 面板可见性状态 |
| `api/commands.ts` | M13 | 所有 invoke() 的唯一入口 |
| `api/events.ts` | M13 | 所有 listen() 的唯一入口 |
| `hooks/useKeyboard.ts` | M05 | 处理 ←/→/Enter/Delete/Backspace/Esc |
| `hooks/useClipboardEvents.ts` | M12 | 订阅 new-record/record-deleted 事件更新 store |

## 4. 后端目录结构（`src-tauri/`）

```
src-tauri/
├── src/
│   ├── main.rs                         # 应用入口，初始化 Tauri Builder，注册插件
│   ├── lib.rs                          # 注册所有 Command，加载开发种子数据
│   ├── error.rs                        # AppError 枚举，实现 serde::Serialize
│   ├── ipc/                            # [M13] IPC 通信模块（后端侧）
│   │   ├── mod.rs                      # 模块导出
│   │   ├── commands.rs                 # #[tauri::command] 函数定义（5个 Command）
│   │   └── events.rs                   # emit() 工具函数（new-record/record-deleted）
│   ├── clipboard/                      # [M01] 粘贴板监听 + 内存存储
│   │   ├── mod.rs                      # 模块导出，ClipboardMonitor trait 定义
│   │   ├── history.rs                  # ClipboardHistory 聚合根
│   │   ├── record.rs                   # ClipboardRecord 实体
│   │   ├── types.rs                    # RecordId、ContentType、PasteMode 值对象
│   │   ├── events.rs                   # ClipboardDomainEvent 领域事件
│   │   ├── store.rs                    # InMemoryStore（内存存储实现）
│   │   ├── repository.rs               # ClipboardRecordRepository trait + InMemory 实现
│   │   ├── monitor.rs                  # ClipboardMonitorService（200ms 轮询）
│   │   ├── filter.rs                   # 去重过滤逻辑（v0.1 仅去重，无黑名单）
│   │   └── test_fixtures.rs            # 测试夹具工厂函数
│   ├── window/                         # [M03] 窗口管理模块
│   │   ├── mod.rs                      # WindowManager trait + Tauri 实现
│   │   └── position.rs                 # 屏幕底部位置计算
│   ├── shortcut/                       # [M04] 快捷键管理模块
│   │   └── mod.rs                      # ShortcutManager，注册 Shift+Cmd+V
│   ├── paste/                          # [M06] 粘贴执行模块
│   │   ├── mod.rs                      # PasteService + PasteExecutor trait
│   │   └── text_strip.rs               # 预留文件（v0.1 暂不使用，v0.2 实现 HTML 剥离）
│   ├── config/                         # [M09] 配置管理（基础版，内存配置）
│   │   ├── mod.rs                      # 模块导出
│   │   └── schema.rs                   # AppConfig 结构体（v0.1 仅包含 max_records 等基础项）
│   └── platform/                       # [M14] 平台抽象（macOS only）
│       ├── mod.rs                      # PlatformClipboard / PlatformKeySimulator trait 定义
│       └── macos.rs                    # macOS 实现（NSPasteboard、CGEventCreateKeyboardEvent）
├── icons/                              # 应用图标
│   ├── 32x32.png
│   ├── 128x128.png
│   ├── 128x128@2x.png
│   ├── icon.icns                       # macOS 应用图标
│   └── icon.png                        # 通用图标
├── Cargo.toml                          # Rust 依赖配置
├── Cargo.lock
├── build.rs                            # Tauri 构建脚本
└── tauri.conf.json                     # 窗口无边框/透明/置顶配置
```

### 4.1 关键文件职责说明（v0.1）

| 文件 | 模块 | 职责 |
|------|------|------|
| `src/main.rs` | — | `fn main()`，创建 Tauri app，启动事件循环 |
| `src/lib.rs` | — | 注册 5 个 Command，初始化 M01/M03/M04/M06/M09/M14 |
| `src/error.rs` | — | `AppError` 枚举（6 个错误码），serde 序列化 |
| `src/ipc/commands.rs` | M13 | 5 个 `#[tauri::command]` 函数实现 |
| `src/ipc/events.rs` | M13 | `emit_new_record()` 和 `emit_record_deleted()` |
| `src/clipboard/history.rs` | M01 | `ClipboardHistory` 聚合，管理记录列表不变式 |
| `src/clipboard/store.rs` | M01 | `InMemoryStore`，内存 CRUD + 自动淘汰 |
| `src/clipboard/monitor.rs` | M01 | 后台 Tokio task，200ms 轮询 `arboard` |
| `src/window/mod.rs` | M03 | 无边框窗口的显示/隐藏/定位到底部 |
| `src/shortcut/mod.rs` | M04 | 注册 Shift+Cmd+V，触发 `WindowManager::toggle()` |
| `src/paste/mod.rs` | M06 | `PasteService::paste()`，5步粘贴流程 |
| `src/config/schema.rs` | M09 | `AppConfig { max_records: 20 }`，v0.1 硬编码默认值 |
| `src/platform/macos.rs` | M14 | `NSPasteboard` 读取文本，`CGEvent` 模拟 Cmd+V |

## 9. 命名规范

### 前端（TypeScript / React）

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| React 组件文件 | PascalCase | `TextCard.tsx`、`EmptyState.tsx` |
| Hook 文件 | camelCase，`use` 前缀 | `useKeyboard.ts` |
| Store 文件 | camelCase，`use` 前缀 | `useClipboardStore.ts` |
| API 封装文件 | camelCase | `commands.ts` |
| 类型文件 | camelCase | `clipboard.ts` |
| 测试文件 | 同名 + `.test` 后缀 | `TextCard.test.tsx` |

### 后端（Rust）

| 类型 | 命名规范 | 示例 |
|------|----------|------|
| 模块目录 | snake_case | `clipboard/`、`window/` |
| 源文件 | snake_case | `monitor.rs`、`history.rs` |
| 模块入口 | 固定 `mod.rs` | `clipboard/mod.rs` |
| struct | PascalCase | `ClipboardRecord`、`InMemoryStore` |
| trait | PascalCase | `ClipboardMonitor`、`PasteExecutor` |
| 函数/变量 | snake_case | `get_recent()`、`max_records` |
| 常量 | SCREAMING_SNAKE_CASE | `DEFAULT_MAX_RECORDS` |
