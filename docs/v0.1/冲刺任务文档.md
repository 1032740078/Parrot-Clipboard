# v0.1 冲刺任务文档

## 1. 文档概述

### 1.1 版本范围

本文档为 **v0.1 MVP 版本**的冲刺任务规划，按模块开发顺序拆解所有可执行的开发任务，并标注依赖关系、预估工作量和验收标准。

### 1.2 v0.1 整体目标

在 macOS 上实现并验证核心使用流程：**复制文本 → 快捷键调出面板 → 方向键选择 → Enter 粘贴**。

- **平台**：仅 macOS
- **内容类型**：仅纯文本
- **存储**：内存存储，重启清空
- **预计周期**：2-3 周

### 1.3 开发批次规划

按模块依赖关系分为 5 个开发批次，每批次内的任务可并行开展。

```
第 1 批：项目基础设施 + 基础模块（无依赖）
    │
    ▼
第 2 批：核心后端功能
    │
    ▼
第 3 批：交互功能
    │
    ▼
第 4 批：主面板 UI
    │
    ▼
第 5 批：集成联调 + 测试 + 验收
```

### TASK-03-1：实现 M01 粘贴板监听服务

**所属批次**：第 3 批
**预估工作量**：1 天
**负责模块**：M01 粘贴板监听
**依赖**：TASK-02-2、TASK-02-3

**任务描述**：

实现 `ClipboardMonitorService`，在后台 Tokio task 中每 200ms 轮询一次粘贴板变化，检测到新文本时更新 `ClipboardHistory`，并通过 IPC 推送事件。

**文件**：`src-tauri/src/clipboard/monitor.rs`

**核心逻辑**：

```rust
async fn poll_loop(&self, app: AppHandle) {
    loop {
        if !self.is_paused.load(Ordering::Relaxed) {
            let current_count = self.platform.change_count();
            let last_count = self.last_change_count.load(Ordering::Relaxed);

            if current_count != last_count {
                self.last_change_count.store(current_count, Ordering::Relaxed);

                if let Ok(Some(text)) = self.platform.read_text() {
                    let events = self.history.write().unwrap().add_record(text, now_ms());
                    for event in events {
                        emit_domain_event(&app, event);
                    }
                }
            }
        }
        tokio::time::sleep(Duration::from_millis(200)).await;
    }
}
```

**所属批次**：第 4 批
**预估工作量**：0.5 天
**负责模块**：M12 前端状态管理
**依赖**：TASK-04-1、TASK-04-2

**任务描述**：

实现 `useClipboardEvents` Hook，订阅后端推送事件，自动更新 store。

**文件**：`src/hooks/useClipboardEvents.ts`

```typescript
export function useClipboardEvents() {
  useEffect(() => {
    const unlistenPromises = [
      onNewRecord(({ record, evicted_id }) => {
        useClipboardStore.getState().addRecord(record);
        if (evicted_id !== undefined) {
          useClipboardStore.getState().removeRecord(evicted_id);
        }
      }),
      onRecordDeleted(({ id }) => {
        useClipboardStore.getState().removeRecord(id);
      }),
    ];

    return () => {
      // 组件卸载时取消所有订阅
      Promise.all(unlistenPromises).then((fns) => fns.forEach((fn) => fn()));
    };
  }, []);
}
```

**集成内容**：

```tsx
// src/App.tsx
import { MainPanel } from './components/MainPanel';
import { useClipboardEvents } from './hooks/useClipboardEvents';
import { useUIStore } from './stores/useUIStore';

export default function App() {
  // 订阅后端事件
  useClipboardEvents();

  const { isPanelVisible } = useUIStore();

  return (
    <MainPanel isVisible={isPanelVisible} />
  );
}
```

**联调验证核心流程**：

1. 复制文本 → 面板中出现记录卡片（后端监听 + 事件推送 + 前端更新）
2. 按快捷键 → 面板弹出（后端快捷键 + 窗口管理 + 前端动画）
3. 方向键切换 → 选中状态变化（前端键盘 + Store 更新）
4. Enter → 粘贴 + 面板关闭（前端事件 → IPC → 后端 PasteService → 模拟按键）
5. Delete → 记录消失 + 事件推送 + 前端同步删除
