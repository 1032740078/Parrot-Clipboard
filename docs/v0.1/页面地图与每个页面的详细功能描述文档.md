# v0.1 页面地图与每个页面的详细功能描述文档

## 1. 文档概述

### 1.1 版本范围

本文档描述 **v0.1 MVP 版本**的完整页面地图，以及每个页面/视图的详细功能规格。

**v0.1 页面范围**：仅包含核心交互所需的最小页面集合。

### 1.2 v0.1 界面清单

| 界面 ID | 名称 | 类型 | 说明 |
|---------|------|------|------|
| P01 | 主面板 | 面板（Panel） | 核心操作界面，快捷键触发 |
| V01 | 文本卡片 | 卡片视图组件 | 文本记录的展示单元 |
| V04 | 空状态视图 | 视图组件 | 无记录时的替代展示 |

**v0.1 不包含**：P01-S（搜索态）、V02（图片卡片）、V03（文件卡片）、P02（设置窗口）、P03（引导页）、P04（托盘菜单）、P05（确认弹窗）

## 2. 页面地图（v0.1）

```
应用启动（macOS，进程后台运行）
    │
    ▼
后台驻留（无 UI，无托盘图标）
    │
    ├── 粘贴板监听线程运行中（后台，无感知）
    │
    └── 用户按下 Shift + Command + V
            │
            ▼
        P01 主面板（屏幕底部弹出）
            │
            ├── 有记录 → V01 文本卡片列表
            │
            └── 无记录 → V04 空状态视图
                    │
                    ├── ← / → 键  →  切换卡片选中状态
                    │
                    ├── Enter 键   →  粘贴并关闭面板
                    │
                    ├── Delete / Backspace  →  删除选中记录
                    │
                    └── Esc / 点击面板外  →  关闭面板
```

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 界面类型 | 无边框悬浮面板（Frameless Window） |
| 触发方式 | 全局快捷键 `Shift + Command + V` |
| 关闭方式 | 按 `Esc`、点击面板外区域、执行粘贴操作后自动关闭 |
| 位置 | 屏幕底部，紧贴屏幕下边缘 |
| 宽度 | 100% 屏幕宽度 |
| 高度 | 固定 220px（v0.1，不含工具栏的简化版） |
| 视觉风格 | 毛玻璃半透明背景（浅色模式），深色模式自动适配 |
| 层级 | 始终置顶（`always_on_top: true`） |
| 是否显示任务栏 | 否（`skip_taskbar: true`） |
| 对应后端模块 | M03 窗口管理、M01 粘贴板监听 |
| 对应前端模块 | M05 主面板 UI、M12 状态管理 |

### 3.2 功能定位

主面板是 v0.1 的**唯一用户界面**，承载所有用户交互。设计目标：

- **弹出即用**：按键后 100ms 内呈现
- **用完即走**：粘贴后自动关闭
- **纯键盘操作**：无需鼠标即可完成全部操作

### 3.3 布局结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐     │
│  │          │  │          │  │          │  │          │  │          │ ···  │
│  │ 文本卡片  │  │ 文本卡片  │  │ 文本卡片  │  │ 文本卡片  │  │ 文本卡片  │     │
│  │ [选中态] │  │          │  │          │  │          │  │          │     │
│  │          │  │          │  │          │  │          │  │          │     │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘  └──────────┘     │
│  ◀ 左右滚动查看更多记录（最多 20 条）                                       ▶ │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

v0.1 去除工具栏，仅包含卡片列表区域，保持最小化界面。

### 3.4 区域说明

**卡片列表区（Card List Area）**

| 属性 | 规格 |
|------|------|
| 排列方式 | 横向排列，最新记录在最左侧 |
| 最大卡片数 | 20 条 |
| 卡片间距 | 12px |
| 列表内边距 | 左右各 16px，上下各 20px |
| 滚动方式 | 水平滚动（overflow-x: auto），鼠标滚轮或方向键驱动 |
| 选中高亮 | 当前聚焦卡片显示高亮边框和背景 |

### 3.5 键盘交互规格

| 按键 | 触发条件 | 动作 | 响应时间要求 |
|------|----------|------|-------------|
| `←` | 面板有焦点，selectedIndex > 0 | selectedIndex - 1，滚动到选中卡片 | < 50ms |
| `→` | 面板有焦点，selectedIndex < records.length - 1 | selectedIndex + 1，滚动到选中卡片 | < 50ms |
| `←` | selectedIndex = 0 | 无操作，不越界 | — |
| `→` | selectedIndex = 末尾 | 无操作，不越界 | — |
| `Enter` | 面板有焦点，有选中记录 | 粘贴选中记录（原格式），面板自动关闭 | < 200ms（端到端） |
| `Delete` | 面板有焦点，有选中记录 | 删除选中记录，选中下一条 | < 50ms（UI 响应） |
| `Backspace` | 面板有焦点，有选中记录 | 同 Delete | < 50ms |
| `Esc` | 面板有焦点 | 关闭面板，不执行操作 | < 300ms（动画完成） |

### 3.6 状态变体

| 状态名 | 触发条件 | 展示内容 |
|--------|----------|----------|
| **正常态** | 有至少 1 条记录 | 文本卡片横向列表，最左侧默认高亮 |
| **空状态** | 历史记录为空 | V04 空状态视图替代卡片列表 |
| **加载态** | 首次打开，正在从存储读取记录 | 骨架屏（v0.1 因内存存储极快，可能无感知） |

### 3.7 动画规格

| 动画 | 方向 | 时长 | 缓动函数 | 触发时机 |
|------|------|------|----------|----------|
| 面板弹出 | 从底部向上滑入（translateY: 100% → 0） | 200ms | `ease-out` | 快捷键触发显示 |
| 面板收起 | 向下滑出（translateY: 0 → 100%） | 200ms | `ease-in` | Esc / 粘贴后 / 失焦后 |
| 卡片选中切换 | 边框颜色 + 背景颜色过渡 | 150ms | `ease-in-out` | 方向键切换选中 |

### 3.8 视觉规格

| 属性 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 背景色 | rgba(255,255,255,0.8) | rgba(28,28,30,0.85) |
| 模糊效果 | `backdrop-filter: blur(40px) saturate(180%)` | 同左 |
| 上边框 | `1px solid rgba(0,0,0,0.08)` | `1px solid rgba(255,255,255,0.08)` |
| 圆角 | 顶部左右各 12px，底部 0 | 同左 |

### 3.9 行为规格（边界情况）

| 场景 | 行为 |
|------|------|
| 面板弹出时粘贴板刚被监听到新内容 | 新记录出现在最左侧，selectedIndex 重置为 0 |
| 面板弹出后无任何操作，用户点击别的应用窗口 | 面板自动收起（失去焦点触发） |
| 应用首次启动，用户立即按快捷键 | 面板弹出，显示空状态（监听刚开始，无记录） |

## 4. V01 文本卡片（Text Card）

### 4.1 基本信息

| 项目 | 说明 |
|------|------|
| 界面类型 | 卡片视图组件（React Component） |
| 归属页面 | P01 主面板卡片列表区 |
| 对应模块 | M05 主面板 UI 模块 |

### 4.2 尺寸规格

| 属性 | 值 | 说明 |
|------|-----|------|
| 宽度 | 240px | 固定宽度 |
| 高度 | 180px | 固定高度 |
| 圆角 | 12px | 四角均匀 |
| 内边距 | 12px | 四边均匀 |

### 4.3 布局结构

```
┌──────────────────────────────┐  ← 宽 240px，高 180px
│ 文本      [来源应用图标 16px] │  ← 顶部标题栏（高 28px）
├──────────────────────────────┤  ← 分隔线 1px
│ N 分钟前                     │  ← 时间戳行（高 20px，12px 字号）
├──────────────────────────────┤
│                              │
│ 内容预览文本（最多 3 行）      │  ← 内容区（高 80px，14px 字号）
│ 超长末尾省略...              │
│                              │
└──────────────────────────────┤
│ N 个字符           ≡ 序号    │  ← 底部元信息栏（高 28px，12px 字号）
└──────────────────────────────┘
```

### 4.4 展示信息详细规格

#### 顶部标题栏

| 元素 | 内容 | 样式 |
|------|------|------|
| 类型标签 | `文本`（固定文字，v0.1 所有卡片均为文本） | 14px，字重 600 |
| 顶部色块背景 | 蓝色（文本类型）：浅色 `#007AFF`，深色 `#0A84FF` | 色块高度 28px |
| 来源应用图标 | v0.1 暂不显示（来源应用获取在 macOS 需额外权限） | — |

#### 时间戳行

| 格式 | 条件 | 示例 |
|------|------|------|
| `刚刚` | created_at 在 60 秒内 | `刚刚` |
| `N 分钟前` | 1 分钟 ≤ 时差 < 60 分钟 | `5 分钟前` |
| `N 小时前` | 1 小时 ≤ 时差 < 24 小时 | `3 小时前` |
| `昨天` | 时差在 24-48 小时内 | `昨天` |
| `N 天前` | 超过 48 小时 | `2 天前` |

时间戳颜色：`Text Secondary`（`#8E8E93`）

#### 内容预览区

| 属性 | 规格 |
|------|------|
| 最大行数 | 3 行 |
| 溢出处理 | CSS `-webkit-line-clamp: 3`，超出显示 `...` |
| 字体大小 | 14px |
| 行高 | 20px（因此内容区最高 60px） |
| 字体颜色 | `Text Primary`（`#000000`/`#FFFFFF`） |
| 空白处理 | `white-space: pre-line`（保留换行符） |
| 字体 | 系统默认字体（`-apple-system, BlinkMacSystemFont`） |

#### 底部元信息栏

| 元素 | 内容 | 位置 |
|------|------|------|
| 字符数 | `N 个字符`（N 为 text_content.len()） | 左侧 |
| 序号标记 | `≡ N`（N 为记录在列表中的显示序号，从 1 开始） | 右侧 |

### 4.5 交互状态规格

| 状态 | 边框 | 背景 | 阴影 | 缩放 |
|------|------|------|------|------|
| 默认态 | `1px solid var(--color-border)` | `var(--color-bg-secondary)` | 无 | 1.0 |
| 悬停态 | `1px solid var(--color-border)` | 同默认 | `0 2px 8px rgba(0,0,0,0.1)` | 1.0 |
| 选中态 | `2px solid var(--color-primary)` | `var(--color-bg-secondary)` | `0 4px 12px rgba(0,122,255,0.2)` | 1.02 |

选中态的背景可叠加 `var(--color-primary)` 10% 透明度：浅色 `rgba(0,122,255,0.08)`，深色 `rgba(10,132,255,0.12)`。

## 6. 页面间状态转换

### 6.1 主面板状态机

```
隐藏态（Hidden）
    │
    │ 快捷键 Shift+Cmd+V
    ▼
弹出动画中（Entering，200ms）
    │
    ▼
可见 - 正常态（Visible, Normal）
    │                           │
    │ 有记录                    │ 无记录
    ▼                           ▼
V01 文本卡片列表             V04 空状态视图
    │
    ├── Enter 键（粘贴）
    │       │
    │       ▼
    │   收起动画中（Exiting，200ms）→ 隐藏态
    │
    ├── Esc 键 / 失焦
    │       │
    │       ▼
    │   收起动画中（Exiting，200ms）→ 隐藏态
    │
    ├── ← / → 键（切换选中）
    │       │
    │       ▼
    │   V01 卡片状态更新（选中项变化）
    │
    └── Delete / Backspace（删除）
            │ 删除后仍有记录
            ▼
        V01 卡片列表更新（删除动画）
            │
            │ 删除后无记录
            ▼
        V04 空状态视图
```

## 7. 响应式适配（v0.1 仅 macOS）

| 屏幕分辨率 | 卡片可见数量（估算） | 说明 |
|-----------|---------------------|------|
| 1280×800（MacBook Air 13") | 约 4-5 张 | 列表可左右滚动 |
| 1440×900（MacBook Pro 13") | 约 5-6 张 | — |
| 1680×1050（MacBook Pro 15"） | 约 6-7 张 | — |
| 2560×1440（外接 27" 显示器） | 约 10 张 | — |

计算方式：`可见数 = floor((屏幕宽度 - 32px 边距) / (240px + 12px))`

不满一张的卡片通过滚动可见（视觉上会被裁切，提示用户可滚动）。

## 8. 无障碍设计（Accessibility）

### v0.1 最低要求

| 要求 | 实现方式 |
|------|----------|
| 键盘完全可操作 | 所有功能均可通过键盘操作，无需鼠标 |
| 颜色对比度 | 文本与背景对比度 ≥ 4.5:1（WCAG AA） |
| 焦点指示器 | 选中卡片有明确的视觉边框高亮 |
| ARIA 角色 | 卡片列表使用 `role="list"`，单卡片使用 `role="listitem"` |
| 选中状态 | 当前选中卡片添加 `aria-selected="true"` 属性 |
