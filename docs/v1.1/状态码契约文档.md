# v1.1 状态码契约文档

## 1. 文档概述

### 1.1 版本范围

`v1.1` 不新增业务错误体系，主要沿用 `v1.0` 状态码。本文件仅列出当前版本会直接涉及的状态码与处理约定。

### 1.2 错误对象格式

```ts
export interface AppErrorPayload {
  code: string;
  message: string;
  details?: Record<string, unknown>;
}
```

### 1.3 错误传递机制

- 后端 `Command` 失败时，统一返回标准错误对象
- 前端根据 `code` 映射用户可读提示
- 自动隐藏属于正常行为，不应以错误码表达

## 2. 当前版本状态码清单

| 状态码 | 名称 | 适用场景 |
|--------|------|----------|
| `SC-001` | `RECORD_NOT_FOUND` | `Command + 数字` 或 `Enter` 对应记录不存在 |
| `SC-008` | `UNSUPPORTED_PLATFORM_FEATURE` | 非 `macOS` 平台误触发当前版本专属快贴能力 |
| `SC-010` | `WINDOW_ERROR` | 主面板显示、隐藏、定位、焦点恢复失败 |
| `SC-013` | `CLIPBOARD_WRITE_ERROR` | 写入系统剪贴板失败 |
| `SC-014` | `KEY_SIM_ERROR` | 模拟粘贴按键失败 |
| `SC-017` | `PERMISSION_REQUIRED` | `macOS` 辅助功能权限缺失，无法完成粘贴 |
| `SC-021` | `INTERNAL` | 未归类的内部错误 |

---

## 3. 状态码详细说明

### SC-001：`RECORD_NOT_FOUND`

- **触发条件**：目标记录已被删除、序号映射失效、运行态与 UI 顺序不一致
- **前端处理**：提示“目标记录不存在，请重新选择”；保持主面板打开

### SC-008：`UNSUPPORTED_PLATFORM_FEATURE`

- **触发条件**：当前平台不支持本版定义的 `Command + 数字` 快贴语义
- **前端处理**：静默降级或提示“当前平台暂不支持快速粘贴”

### SC-010：`WINDOW_ERROR`

- **触发条件**：主面板定位、覆盖 `Dock`、显示 / 隐藏、焦点恢复失败
- **前端处理**：提示“主面板窗口操作失败，请重试”

### SC-013：`CLIPBOARD_WRITE_ERROR`

- **触发条件**：粘贴前写入系统剪贴板失败
- **前端处理**：提示“写入系统粘贴板失败，请重试”

### SC-014：`KEY_SIM_ERROR`

- **触发条件**：组合键模拟粘贴失败
- **前端处理**：提示“模拟粘贴失败，请检查辅助功能权限”

### SC-017：`PERMISSION_REQUIRED`

- **触发条件**：`macOS` 下执行 `Enter`、`Shift+Enter`、`Command + 数字` 时缺少辅助功能权限
- **前端处理**：保持主面板打开，提示并提供权限引导入口

### SC-021：`INTERNAL`

- **触发条件**：非预期内部错误
- **前端处理**：提示通用错误，并记录日志

## 4. Command / Event 对应矩阵

| 接口 | 可能状态码 |
|------|------------|
| `paste_record` | `RECORD_NOT_FOUND`、`CLIPBOARD_WRITE_ERROR`、`KEY_SIM_ERROR`、`PERMISSION_REQUIRED`、`INTERNAL` |
| `hide_panel` | `WINDOW_ERROR`、`INTERNAL` |
| `get_runtime_status` | `INTERNAL` |
| `system:panel-visibility-changed` | 正常情况下不携带错误码；若广播失败仅记录日志 |

---

## 5. 前端统一处理约定

- `focus_lost` 自动隐藏是正常路径，**禁止弹出错误提示**
- `Command + 数字` 快贴失败时，主面板保持可见，方便用户重试或改用 `Enter`
- `RECORD_NOT_FOUND` 与 `PERMISSION_REQUIRED` 都不应关闭主面板
- 所有错误日志禁止带出完整敏感正文，只允许记录 `record_id`、类型、触发键和错误码

---

## 6. 成功响应约定

- 快贴成功：隐藏主面板、恢复目标应用焦点、不额外弹成功 Toast
- 仅快选成功：只更新高亮与滚动位置，不触发任何后台命令
- 自动隐藏成功：同步 `panel_visible=false`，不提示、不打断用户当前应用操作
