# v1.1 BDD 自动化验收文档

## 1. 文档概述

### 1.1 版本范围

本文档仅覆盖 `v1.1` 当前版本的自动化验收场景，不展开 `v1.0` 已完成能力的完整回归清单。

### 1.2 BDD 编写规范

- 使用 `Given / When / Then / And` 四段式表达
- 场景标题优先描述用户感知结果
- 每个场景只验证一个主结果，复杂链路拆成多个场景
- 若包含平台差异，标题中显式标记 `macOS`

---

## 2. Feature：主面板显隐一致性（Panel Visibility）

### 场景 BDD-11-01：主面板失去焦点后自动隐藏

- **Given** 用户已经通过全局快捷键打开主面板
- **And** 主面板当前拥有前台焦点
- **When** 用户点击其他应用窗口或切换到外部应用
- **Then** 主面板会在 `100ms` 内自动隐藏
- **And** 前端与后端的 `panel_visible` 状态保持一致

### 场景 BDD-11-02：自动隐藏后再次呼出可正常工作

- **Given** 主面板已经因为失焦被自动隐藏
- **When** 用户再次通过全局快捷键呼出主面板
- **Then** 主面板重新显示在最前
- **And** 左右切换、数字快选和 `Enter` 粘贴仍然可用

---

## 3. Feature：Dock 覆盖与尺寸恢复（Dock Overlay）

### 场景 BDD-12-01：底部 Dock 场景下主面板恢复原始高度

- **Given** 用户运行在 `macOS` 且 `Dock` 位于底部
- **When** 用户呼出主面板
- **Then** 主面板高度为 `220px`
- **And** 主面板底边贴合物理屏幕底部
- **And** 底部卡片内容完整可见，没有被裁剪

### 场景 BDD-12-02：侧边 Dock 场景下主面板仍覆盖 Dock

- **Given** 用户运行在 `macOS` 且 `Dock` 位于左侧或右侧
- **When** 用户呼出主面板
- **Then** 主面板宽度按物理屏幕宽度计算
- **And** 主面板不会因为 `Dock` 占位而缩窄或错位

### 场景 BDD-12-03：多显示器切换后主面板仍显示在目标屏幕底部

- **Given** 用户连接了两块缩放比例不同的显示器
- **And** 光标当前位于副屏
- **When** 用户呼出主面板
- **Then** 主面板显示在副屏底部
- **And** 主面板仍采用覆盖 `Dock` 的定位策略

## 4. Feature：快速粘贴（Quick Paste）

### 场景 BDD-13-01：`Command + 数字` 直接粘贴对应记录

- **Given** 主面板当前已打开且存在至少 `3` 条记录
- **When** 用户按下 `Command + 3`
- **Then** 第 `3` 条记录会被直接粘贴
- **And** 主面板会自动隐藏
- **And** 该记录会更新为最近使用状态

### 场景 BDD-13-02：单独数字键仍然只是快选

- **Given** 主面板当前已打开且存在至少 `3` 条记录
- **When** 用户按下数字键 `3`
- **Then** 第 `3` 条记录会被选中
- **And** 不会立即触发粘贴

### 场景 BDD-13-03：权限缺失时快速粘贴被阻止并提示

- **Given** 用户运行在 `macOS`
- **And** 辅助功能权限状态为 `missing`
- **And** 主面板当前已打开
- **When** 用户按下 `Command + 1`
- **Then** 不会执行实际粘贴
- **And** 主面板保持打开
- **And** 用户会看到明确的权限提示

---

## 5. Feature：选中项可见性保障（Ensure Visible）

### 场景 BDD-14-01：左右切换超出可见区时自动滚动

- **Given** 主面板存在一组超出当前视口宽度的记录
- **And** 当前选中项位于可见区边缘
- **When** 用户连续按下 `ArrowRight`
- **Then** 当下一个选中项超出可见区时，列表会自动滚动
- **And** 新选中项保持可见

### 场景 BDD-14-02：数字快选跳转远端记录时自动滚动

- **Given** 主面板存在至少 `9` 条记录
- **And** 第 `9` 条记录初始不在可见区域内
- **When** 用户按下数字键 `9`
- **Then** 第 `9` 条记录被选中
- **And** 列表自动滚动到该卡片可见

### 场景 BDD-14-03：虚拟滚动场景下自动滚动不丢失高亮

- **Given** 主面板加载了 `40+` 条记录并启用虚拟滚动
- **When** 用户使用左右键切换到视口外记录
- **Then** 虚拟列表会自动滚动并渲染目标卡片
- **And** 高亮项、实际选中索引和后续粘贴目标保持一致

## 6. 非功能需求验收场景（Performance / Reliability）

### 场景 BDD-NFR-11：失焦自动隐藏无明显迟滞

- **Given** 用户已打开主面板
- **When** 用户切换到其他应用
- **Then** 主面板应在体感上立即收起
- **And** 不出现“主面板短暂停留在最前层”的明显迟滞

### 场景 BDD-NFR-12：快速粘贴不会引入额外误触

- **Given** 主面板已打开
- **When** 用户按下 `Command + 数字`
- **Then** 不会同时触发“快选 + 再次粘贴”双重动作
- **And** 不会把错误记录粘贴到目标应用

|------|----------|

## 8. 通过标准

- `P0` 场景全部通过
- `macOS` 专属场景至少在 `Dock` 底部 / 左侧 / 右侧三种布局下跑通
- 快贴与自动滚动场景不能引入已有 `Enter` / `Shift+Enter` 回归
