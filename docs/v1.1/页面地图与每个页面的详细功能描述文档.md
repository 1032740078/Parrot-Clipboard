# v1.1 页面地图与每个页面的详细功能描述文档

## 1. 文档概述

### 1.1 版本范围

`v1.1` 不新增独立页面，主要调整 **主面板** 的交互与窗口行为；其他页面沿用 `v1.0` 信息架构，仅补充与当前版本功能的关系说明。

### 1.2 v1.1 界面清单

| 页面 / 视图 | 是否变更 | 说明 |
|-------------|----------|------|
| `P01` 主面板 | 是 | 本版核心改动页 |
| `P02` 设置窗口 | 否 | 结构沿用 `v1.0` |
| `P03` 关于页 | 否 | 结构沿用 `v1.0` |
| `P04` 权限引导弹窗 | 小幅关联 | 作为快贴失败时的降级入口 |
| `P05` 托盘菜单 | 否 | 无信息架构变化 |
| `P06` 通用确认弹窗 | 否 | 无结构变化 |

## 3. P01 主面板（Main Panel）

### 3.1 基本信息

| 项目 | 说明 |
|------|------|
| 页面定位 | 高频复制 / 粘贴的主工作界面 |
| 打开方式 | 全局快捷键、托盘入口 |
| 关闭方式 | `Esc`、粘贴成功、失去焦点、显式隐藏 |
| 本版优先级 | `P0` |

### 3.2 v1.1 变化点

- 主面板失去焦点时自动隐藏
- 面板调整为可完整展示卡片的固定高度 `304px`
- 面板直接覆盖 `Dock`，不再依赖“缩短高度”规避
- 支持 `Command + 1~9` 直接粘贴
- 左右切换与数字快选命中视口外卡片时自动横向滚动

### 3.3 区域说明

| 区域 | 说明 |
|------|------|
| 标题区 | 展示“最近记录”和辅助说明 |
| 权限提示区 | 权限缺失时出现，阻止快贴 / 粘贴无反馈 |
| 卡片列表区 | 横向卡片列表，负责高亮、滚动、自动可见性保障 |
| 快捷键提示区 | 展示 `Enter`、`Shift+Enter`、`1~9`、`Command+1~9`、`Esc` |

### 3.4 状态变体

| 状态 | 描述 |
|------|------|
| 正常态 | 可浏览、可快选、可快贴 |
| 权限受限态 | 可浏览 / 选择，不可执行粘贴与快贴 |
| 长列表态 | 启用虚拟滚动，自动滚动仍需保证选中项可见 |
| 自动隐藏态 | 失焦后立即退出界面 |

### 3.5 关键交互规格

| 交互 | 结果 |
|------|------|
| `ArrowLeft / ArrowRight` | 切换选中项；若超出可见区则自动滚动 |
| `1~9` | 仅选中对应序号记录 |
| `Command+1~9` | 直接粘贴对应序号记录并隐藏主面板 |
| `Esc` | 隐藏主面板 |
| 点击外部应用 | 主面板自动隐藏 |

## 6. P04 权限引导弹窗（Permission Guide）

### 6.1 触发条件

- 用户按 `Enter`
- 用户按 `Shift+Enter`
- 用户按 `Command + 数字`
- 且当前 `macOS` 辅助功能权限缺失

### 6.2 内容组成

- 权限缺失说明
- 打开系统设置入口
- 授权后重试说明

### 6.3 行为规格

- 快贴失败时优先提示原因，不允许无反馈
- 用户关闭引导后仍保留主面板上下文，方便再次选择

## 9. 视图组件说明

### 9.1 `V12` 快贴提示项

- 位于主面板底部快捷键提示区
- 文案建议：`⌘+1~9 快贴`

### 9.2 `V13` 自动隐藏行为

- 不是独立 UI 组件，而是主面板运行态规则
- 不额外弹 Toast，不打断用户当前应用

### 9.3 `V14` 选中项自动可见

- 不是新页面，而是 `CardList` 的关键交互约束
- 需要同时覆盖普通横向列表与虚拟滚动列表

## 10. 页面间状态转换

| 起点 | 条件 | 终点 |
|------|------|------|
| 主面板可见 | 用户点击外部应用 | 主面板隐藏 |
| 主面板可见 | 用户按 `Esc` | 主面板隐藏 |
| 主面板可见 | 用户按 `Command+数字` 且粘贴成功 | 主面板隐藏 |
| 主面板可见 | 用户按 `Command+数字` 且权限缺失 | 权限引导弹窗 / 保持主面板 |
| 主面板可见 | 用户点击关于 | 关于页 |

---

## 11. 可访问性要求

- 快捷键提示区必须明确区分“快选”和“快贴”
- 高亮卡片需要有足够的边框 / 阴影对比，便于用户确认当前目标
- 自动滚动后不得丢失焦点感知
